HS_pkgUpdateCacheOnServer

Description: updates the package list on the server ( for editing package selections ) and saves it gzipped in /m23/data+scripts/distr/halfSister/packages-$packagesourcename-$arch.gz
Parameter:
  • packagesourcename: name of the package source
  • force: force the update (or not)
  • arch: CPU architecture of the client.
Returns:
  • The name of the package cache.
