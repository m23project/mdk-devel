CFDiskIO::getCurrentDiskSizesForDefinedDiskSizes

Description: Gets the disk devices and their sizes, if the client is a defined client.
Parameter:
  • overwrite: Set to true, if the disk devices and their sizes should be saved even if the client is NOT a defined client.
Returns:
  • Array with disk devices and their sizes, if the client is a defined client, otherwise false.
