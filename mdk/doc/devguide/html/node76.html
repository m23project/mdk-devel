Turn off m23 script debugging

You have two possibillities to deactivate the script debugging: the first solution is temporary the second is usefuly to stop debugging for a longer time.
  1. Edit the row for your client in the "remotevar" table. Set the value from the "value" field to "0".
  2. Delete the row of your client in the "remotevar" table.
