CClientLister::isVisibleByPingableFilter

Description: Checks, if only clients that can (not) be pinged should be shown and if the given client matches the filter rule.
Parameter:
  • CClientO: Object to the client to check.
Returns:
  • true, if there is no pingable filter rule, or if the client matches the filter rule.
