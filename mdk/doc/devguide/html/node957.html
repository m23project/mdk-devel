CPool::__construct

Description: Constructor for new CPool objects that loads existing pools or creates a new basic pool.
Parameter:
  • poolName: Name of the pool (if a pool with the given name exists = load)
  • poolType: Type of the pool (POOL_TYPE_CD or CPoolLister::POOL_TYPE_DOWNLOAD).
  • poolArch: Architecture of the pool (POOL_ARCH_I386 or CPoolLister::POOL_ARCH_AMD64).
