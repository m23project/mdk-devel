MSR_statusBar

Description: Sets new percent value and/or new status text.
Parameter:
  • percent: Percent value to write into the DB (may be empty, if it should not be changed).
  • statustext: A text message that should be shown under the status bar and written to the DB (may be empty, if it should not be changed).
