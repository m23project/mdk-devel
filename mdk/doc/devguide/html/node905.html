CObjectStorage::__construct

Description: Constructor for new CObjectStorage objects. The object saves/updates or loads an object.
Parameter:
  • ident: Identifier for objects belonging to e.g. a client.
  • classOrObject: Class name of the object or the object itself.
  • readFromDBIfObjectExists: Set to true, if an existing object should read from the DB, even if an object is given.
