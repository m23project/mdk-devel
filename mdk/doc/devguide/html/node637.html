CFDiskIO::fdiskGetEntry

Description: Gets an entry to from the fstab array. Can be called more times to get all entries.
Parameter:
  • dev: Device to mount (e.g. /dev/hda1)
  • mountpoint: Location where to mount the device (e.g. /mnt/hda1)
  • parameter: Mount parameter.
Returns:
  • true, if an entry to from the fstab array could be fetched from the array, otherwise false.
