CFDiskBasic::createInstallPartition

Description: Creates and formats an installation partition and makes it bootable. The created partition is stored in the client parameters as installation partition.
Parameter:
  • diskDev: The disk device (e.g. /dev/hda) where the installation partition should be created on.
  • instStart: Start position of the installation partition (in MB).
  • instEnd: End position of the installation partition (in MB).
