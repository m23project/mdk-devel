AUTOTEST_replaceConstantsInString

Description: Searches for variable declarations in a string and replaces them with the values of according constants.
Parameter:
  • in: Input string. eg. "name=$VM_NAME" may become "name=debianVM"
Returns:
  • The input string with replaced variable declarations.
