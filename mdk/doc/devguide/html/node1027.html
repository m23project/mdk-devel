CRON_addJobDayly

Description: Runs a command every day at a specified time.
Parameter:
  • min: Minute or * for every minute to execute the cron job. (0-59)
  • hour: Hour or * for every hour to execute the cron job. (0-23)
  • user: The user the cron job should be run under.
  • cmd: The command to exectute.
  • identifier: A string to identify the cron entry (for deletion)
  • runInScreen: Set to true if the command should be run in a screen.
