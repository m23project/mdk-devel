CFDiskBasic::createPartition

Description: Creates a new partition on a disk (if possible).
Parameter:
  • dev: The device (e.g. /dev/sda).
  • start: Start position (in MB) of the partition to create.
  • end: End position (in MB) of the partition to create.
  • type: type of the partition (CFDiskIO::PT_PRIMARY, CFDiskIO::PT_EXTENDED, CFDiskIO::PT_LOGICAL)
  • bootable: Set to true, if the partition should be made bootable.
Returns:
  • The physical partition number of the newly created partition.
