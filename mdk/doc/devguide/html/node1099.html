VM_startVM

Description: Starts a virtual machine.
Parameter:
  • type: VM_SW_VBOX for VirtualBox OSE
  • vmName: Name of the VM.
  • vnc: Set to true if the VM should be accessible since the booting via VNC.
Returns:
  • BASH code to start a virtual machine.
