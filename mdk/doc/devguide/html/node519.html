CFDiskGUI::updateDiskLayout

Description: Updates the disk layout for a given disk, if it is not in the cache.
Parameter:
  • vDisk: Internal disk number (for accessing the disk information in the array).
Returns:
  • true, if the disk layout was calculated, false, if it is in the cache
