VM_insertBootISO

Description: Inserts a bootable ISO into a VM.
Parameter:
  • type: VM_SW_VBOX for VirtualBox OSE
  • vmName: Name of the VM.
  • iso: ISO file with full path.
Returns:
  • BASH code to insert a bootable ISO into a VM.
