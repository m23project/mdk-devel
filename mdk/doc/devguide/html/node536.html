HELPER_grubMd5Crypt

Description: Encrypts a password to the MD5 hash as expected by grub.
Parameter:
  • password: Password to encrypt.
  • length: Length of the salt.
Returns:
  • Encrypted password in grub style or false if MD5 hash function isn't available.
