CClient::getActionString

Description: Generates the action string (e.g. an URL for GET)
Parameter:
  • actionString: Action string that may contain CLIENT_NAME (will be replaced by the name of the client) and CLIENT_ID (will be replaced by the ID of the client).
Returns:
  • Changed (or unchanged) action string.
