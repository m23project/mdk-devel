CClient::setKeyValueStore

Description: Sets a value in the key value store of the client.
Parameter:
  • key: Name of the key.
  • value: The value to store under the key.
  • check: An optional variable firewall check constant or rule or a function for check the validity of the value.
  • errorMsg: An error message to give out, if the value fails the check.
