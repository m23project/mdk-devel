usage 

genFstab <splitter> <rootDeviceNr>
  • splitter: is the splitter for the parted numbers . Select '.' for english and ',' for german version. You can find out the language setting very easy: the messages printed in the console have the same language that will used for the splitter.
  • rootDeviceNr: the number of the root device, select '3' for hda3.
