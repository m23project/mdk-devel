CPool::downloadDebsFromClient

Description: Checks, if all pre-requirements for downloading packages to the pool are satisfied. Then starts the routine to download the packages directly from the client.
Parameter:
  • clientName: Name of the client to download the packages from.
  • returnCommands: If set to true, the commands for downloading the packages will be returned instead of executed.
Returns:
  • true, if the download was started otherwise false.
