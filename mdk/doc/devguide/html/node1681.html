HS_hookEndAfterChroot

Description: Scripts that should be run at the end of the afterChroot.
Parameter:
  • rootDevice: Device with partition where the os will be installed (e.g. /dev/hda1).
  • bootDevice: Device (with partition) where the bootmanager will be installed (e.g. /dev/hda).
  • bootDeviceFS: Filesystem of the root device.
