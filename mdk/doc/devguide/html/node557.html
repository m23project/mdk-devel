CFDiskIO::getpDiskAndpPartFromDev

Description: Splits a device (e.g. /dev/hda1) in the physical disk (/dev/hda) and the partition number (1).
Parameter:
  • dev: The device to partition (e.g. /dev/hde1)
  • pDisk: The parameter, the physical disk is written to (e.g. /dev/hda).
  • pPart: The parameter, the physical partition number (e.g. 1) or false (if there is no number in the dev) is written to.
  • ignoreMD: Set to true, if a "partition number" from a MD should be received.
