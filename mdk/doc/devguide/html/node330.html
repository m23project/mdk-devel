CClientLister::isMatchingClientPresent

Description: Checks if at least one clients with a given key-value-combination is found.
Parameter:
  • key: The key to search for (e.g. office)
  • val: The value to search for (e.g. home)
Returns:
  • true, if at least one client is found.
