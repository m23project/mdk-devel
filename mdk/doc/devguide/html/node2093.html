array_keysSearch

Description: Searches for a key in an associative array that matches a regular expression.
Parameter:
  • array: Array to search in.
  • expr: Regular expression for searching the keys (e.g. "/^BUT_action/").
Returns:
  • : The first found key that matches the expression or false if no matching key was found.
