DISTR_getDebianUserGroups

Description: Returns the default groups an user of a Debian system should be in.
Parameter:
  • arrayOrSeparator: If set to true, the groups are returned as array, otherwise the value is used as separator character.
Returns:
  • Default groups as array or as string, where the groups are separates by a separator character.
