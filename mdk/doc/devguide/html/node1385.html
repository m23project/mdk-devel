PKG_updatePackageInfo

Description: Updates the package information to make it searchable.
Parameter:
  • distr: the distribution name
  • packagesource: name of the package source
  • force: set to true if you want to update the package info and the time is not over
  • arch: Architecture to get package infos for.
  • changeTime: Writes the time when the status file was last modified to.
