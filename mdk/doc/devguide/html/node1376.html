DISTR_geti18nValue

Description: returns a value of the info file and tries to get it in the selected language. if it doesn't exist it is returned in the default language
Parameter:
  • lang: language as 2 letter code (e.g. de)
  • variable: the name of the variable you want to get (e.g. Description)
  • values: the associate array of the info.txt values
