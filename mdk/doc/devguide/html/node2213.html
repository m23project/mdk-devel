PKG_getDuplicatedIdenticalJobs

Description: Returns an array with a list of all jobs that are identical by package, priority, status and params.
Parameter:
  • clientName: Name of the client.
  • omitFirstID: Set to true if the first ID of each duplicate jobs block should be omitted.
Returns:
  • Array with a list of all jobs that are identical by package, priority, status and params.
