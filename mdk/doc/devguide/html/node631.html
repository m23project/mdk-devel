CFDiskIO::getvrDevNrByrDev

Description: Returns the (virtual) number of the given device building the RAID.
Parameter:
  • vrDisk: Virtual (internally used) disk number of the RAID.
  • dev: Device to search in the RAID.
Returns:
  • Virtual number of the given device building the RAID or false, if the device is not part of the RAID.
