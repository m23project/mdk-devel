PLG_showDownloadStatus

Description: downloads or copies the plugin to the temp dir
Parameter:
  • url: the place where to get the plugin file from. following transport protocols are allowed: http, ftp and local files. e.g. "http://myserver.de/test.m23plg" is avalid url.
  • tempDir: where to store the plugin file temporary
  • fileName: file name with whole path to the plugin file
