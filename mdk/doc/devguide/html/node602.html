HELPER_getNewLogLines

Description: Gets the last (new) lines of a (growing) log file.
Parameter:
  • log: Name of the log file.
  • sessionPrefix: Prefix for storing the last read line number in the session.
  • filterFunction: Function to filter the lines before adding them to the output. The function gets the unfiltered string as input and returns the filtedred version.
Returns:
  • UTF8-encoded new lines of the log file.
