CClient::setRootPassword

Description: Sets the root password for the client.
Parameter:
  • rootPassword: The (encrypted) root password to set.
  • cryptRootPw: set to true, if the password should be encrypted or false, if it's already encrypted.
Returns:
  • true, if the root password is valid.
