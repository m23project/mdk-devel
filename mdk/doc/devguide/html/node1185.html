CLCFG_setAuthorized_keys

Description: sets the ssh authorized_file for remote login into the clients
Parameter:
  • serverIP: IP of the server
  • pathToKeyFile: path on the server where to get the key file from
  • acceptyDSSKeys: Set to true, if SSH-DSS keys should be re-enabled
