CChecks::genericCHECK_FW

Description: Generic checking routine, that checks if an input value matches a rule.
Parameter:
  • rule: CHECK_FW rule to check the input value with.
  • val: Value to check.
  • errorMsg: Error message to add, if the checking fails.
Returns:
  • true, if the input value matches the rule otherwise false.
