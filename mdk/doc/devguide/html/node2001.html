MAIL_attach

Description: Attaches a file to the message body of the mail and changes the mail header.
Parameter:
  • file: The file with full path to attact.
  • header: The original header that will be modified.
  • message: The text message.
  • fileName: Alternate file name for the attacement as it should be seen by the eMail client.
