CFDiskIO::getRaidDevsBuildingDisk

Description: Returns an array with all disk or partition devices building the RAID of the disk (if it is a RAID).
Parameter:
  • vrDisk: Virtual (internally used) disk number.
Returns:
  • Array with all disk or partition devices building the RAID of the disk (if it is a RAID) or empty array if the disk is no RAID.
