PKG_getKernels

Description: Generates an associative array with the available kernels for an architecture and distribution as keys and values, where kernels containing the term 'generic' are at the front.
Parameter:
  • distr: the distribution name
  • packagesource: name of the package source
  • arch: Architecture to get package infos for.
Returns:
  • Associative array with the available kernels for an architecture and distribution as keys and values with kernels containing the term 'generic' at the front, sorted by length and then by alphabet.
