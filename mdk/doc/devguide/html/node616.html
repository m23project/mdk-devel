CFDiskIO::getPossiblePartitionTypesBetween

Description: Get a list of partition types that can be created in a range on a disk.
Parameter:
  • vDisk: Internal disk number (for accessing the disk information in the array).
  • start: Start position (in MB) of the partition to create.
  • end: End position (in MB) of the partition to create.
Returns:
  • List of partition types that can be created in a range on a disk.
