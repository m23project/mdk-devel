CFDiskBasic::createPartitionJob

Description: Generates a partition add job and adds it to the list of partition steps.
Parameter:
  • dev: selected device (e.g. /dev/hda)
  • start: start point of the partition (in MB)
  • end: end point of the partition (in MB)
  • type: type of the partition (primary, logical)
  • pPart: number of the device (e.g. 1 with /dev/hda1)
