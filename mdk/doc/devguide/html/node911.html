CObjectStorageManager::getObjectMethodReturnValue

Description: Checks, if an objects implements a method, calls it and returns the return value. If the method is unimplemented, an error value will be returned.
Parameter:
  • object: Object to use.
  • fkt: Name of the function (method)
  • error: Error value to return, if the method is unimplemented.
Returns:
  • Return value of the implemented method or given error value if unimplemented.
