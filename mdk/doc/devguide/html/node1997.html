MAIL_gpgCheckKey

Description: Checks, if a GPG key ID has a valid private or public key.
Parameter:
  • gpgID: ID of the private GPG key.
  • privateKey: If set to true, the validity as private key is checked.
Returns:
  • : true, if the GPG key is valid.
