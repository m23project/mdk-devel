MAIL_gpgMail

Description: Encrypts a message with GPG for a given eMail address.
Parameter:
  • message: The message text to encrypt.
  • eMail: The eMail address to search the GPG key for and to encrypt to.
Returns:
  • : Encrypted message or false if there were errors.
