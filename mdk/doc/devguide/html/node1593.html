FDISK_getDriveInfoString

Description: Generates an info string, that shows information about the device name of the drive and bolonging to a RAID.
Parameter:
  • vDev: Virtual (internally used) device number.
  • param: parameter string containing status informations about the harddisks
  • fstabA: Associative array with fstab information.
Returns:
  • Info string.
