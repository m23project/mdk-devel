PKG_fastGetNewInstalledPackages

Description: Gets new installed packages by comparing status files (before and after the installation).
Parameter:
  • oldStatusFile: File with the list of all installed packages before the installation of the new packages
  • newStatusFile: File with the list of all installed packages after the installation of the new packages
  • storeFile: File name to store the list of new installed on the client.
