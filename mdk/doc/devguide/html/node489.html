CFDiskGUI::showFreeSpaceBarBlock

Description: Shows a block for with free space in the bar visualising the partitioning of a disk.
Parameter:
  • pDisk: The physical device name of the disk (eg. /dev/sda)
  • start: Start point of the partition (in MB)
  • end: End point of the partition (in MB)
  • width: The width of the block in percent (without
  • type: Type of the partition.
  • addJavaScript: Set to true to add JavaScript code that calls the JS function emptySpace(), if empty parts of the drive are clicked, selectPartition(), if a partition is clicked and showPartTable(), if the mouse is over the bar.
