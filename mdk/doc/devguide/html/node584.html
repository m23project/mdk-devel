CFDiskIO::getBiggestValueOf

Description: Gets the biggest value from all partitions of a given type.
Parameter:
  • vDisk: Virtual (internally used) disk number.
  • partType: Type of the partition (CFDiskIO::PT_PRIMARY, CFDiskIO::PT_EXTENDED, CFDiskIO::PT_LOGICAL, CFDiskIO::PT_EFI)
  • varType: Variable in the associative array ($this-wantedPartitioning[$vDisk][$vPart][$varType]) to check.
Returns:
  • Biggest value from all partitions of a given type.
