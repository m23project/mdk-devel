CLIENT_getSSHKeyorPasswordOptions

Description: Generates SSH options for using sshpass (password given) or SSH public key (no password given).
Parameter:
  • password: Password to use, if it's not NULL or empty.
  • sshPass: sshpass commands are written here, if a password is given.
  • sshPubkeyOnlyOptions: SSH options are written here, if no password is given.
