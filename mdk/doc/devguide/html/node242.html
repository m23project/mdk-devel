CChecks::checkLogin

Description: Checks if the given login is valid.
Parameter:
  • login: The login name to check.
  • allowEmpty: Set to true, if empty logins should be allowed (e.g. when read from an LDAP server)
Returns:
  • true, if the login name is valid otherwise false.
