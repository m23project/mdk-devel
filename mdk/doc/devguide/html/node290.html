CLIENT_desasterRecovery

Description: recover a client: all client jobs are done again, status is set to 0
Parameter:
  • clientName: name of the client
  • addInstallRemovalJobs: If set to true, the names of all installed packages will be combined to a m23normal and all revomed to a m23normalRemove job.
  • addShutdownOrRebootPackage: If set to true, a shutdown or reboot package will be added.
