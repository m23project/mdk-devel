CFDiskBasic::EFItypeAndGUIDJob

Description: Sets EFI boot partition type and GUID for the EFI boot partition and adds it to the list of partition steps.
Parameter:
  • dev: Disk with the EFI partition (e.g. /dev/hda)
  • pPart: Number of EFI partition, minor number in parted
