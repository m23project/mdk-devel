KDE_installLoginManager

Description: Installs the KDE/Trinity login manager KDM.
Parameter:

  • ver: KDE version 3 or 4 (minor releases are depending in the on the used distribution)
  • trinity: Set to true, if the Trinity Desktop should be configured.
