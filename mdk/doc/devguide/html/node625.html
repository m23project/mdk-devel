CFDiskIO::getPartDevs

Description: Returns an array with the partitions (/dev/hda1, /dev/hda2, ...) of a disk or all disks.
Parameter:
  • vDisk: Virtual (internally used) device number or -1, if all partitions on all devices should be listed.
  • excludeType: type of partitions, not to return.
  • includeTypes: Array with types of partitions, to return.
Returns:
  • Array with matching partitions.
