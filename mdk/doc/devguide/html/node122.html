usage

log2db <dbuser> <dbpassword> <serverIP> <clientname> <logfile>
  • dbuser: database username to connect
  • dbpassword: correct password for this user
  • servrIP: ip of the database server
  • clientname: name of the current client, that wants to send the logile
  • logfile: file you want to log
