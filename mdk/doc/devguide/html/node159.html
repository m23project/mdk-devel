CAutoTest::isAnswerFoundInTriggerResult

Description: Checks, if a trigger result contains an answer.
Parameter:
  • triggerResult: A text (eg. extracted by OCR or from HTML source) that may contain answers.
  • answer: A normal string or an $I18N_ variable. In case of a variable all translations will be searched in the trigger result. If it begins with "!", the answer should NOT be found in the result.
Returns:
  • true, if the answer (or its translation) was (NOT) found in the trigger result.
