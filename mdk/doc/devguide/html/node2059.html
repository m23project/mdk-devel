MSR_genericSendCommand

Description: Generates a generic command for sending information from the client to the server.
Parameter:
  • type: Type of the message understood by MSR_decodeMessage
  • params: Parameters to send with POST to the server in the form of "var1=val1&var2=val2&var3=val3..."
  • wgetParams: Extra parameters for wget.
