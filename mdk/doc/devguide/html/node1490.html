PKG_downloadPool

Description: Generates commands to download packages from a sources list to a directory.
Parameter:
  • destDir: the directory the packages should be downloaded to
  • sourceslist: package source list
  • packagesArr: array of the lists that contain packages to download (seperated by blanks)
  • arch: CPU architecture to download the packages for.
  • release: Select the Debian/Ubuntu suite (squeeze, sarge, sid, precise).
