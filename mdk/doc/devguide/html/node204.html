CAutoUpdate::getType

Description: Gets the type of a set as constant value or in translated form.
Parameter:
  • id: ID of the set.
  • translate: If set to true, the type will be translated to a human readable form.
Returns:
  • Type of a set as constant value or in translated form or false, if no type is set or it is invalid.
