CClientLister::setVisibleByPingableFilter

Description: Sets the pingable filter rule (show only clients that can (not) be pinged or all clients).
Parameter:
  • filterState: true (for showing only pingable clients), false (for not pingable clients) or null for all clients (filter deactivated).
