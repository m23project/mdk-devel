CPackageStatusCompare::showStatusRow

Description: Gets the file name of a package status file. If a client name is give, a temporary package status file will be written.
Parameter:
  • left: Left package name.
  • leftVersion: Left version of the package.
  • leftStatus: Left status of the package.
  • type: diff type.
  • right: Right package name.
  • rightVersion: Right version of the package.
  • rightStatus: Right status of the package.
Returns:
  • Array with, the package file name, the version and the status.
