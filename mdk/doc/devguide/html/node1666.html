GRP_listAllClientsInGroups

Description: returns an array with all client names contained in the groups
Parameter:
  • groupNames: the names of the groups stored in an array
  • withAutoUpdateJob: If set to true, only clients with auto update job will be listed.
Returns:
  • Array with the found client names.
