BT_status

Description: Shows status information about the (maybe) running Bittorrent tracker and initial client.
Parameter:
  • return: If set to true, the result will be returned otherwise shown.
  • nl2br: If set to true, the ASCII line breaks will be converted to HTML line breaks.
Returns:
  • : Status information, if $return == true.
