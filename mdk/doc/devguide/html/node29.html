Documentation

  • Translate all new and changed texts (/m23/inc/help/* and /m23/inc/i18n/*)
  • Fretch and merge the files via /mdk/m23helper/gitCloneAndMeldTranslations .
  • Check for missing I18N variables. Run /mdk/doc/manual/bin/checkForMissingi18n.sh
  • Manual
    • Check, if all parts seem to be there. Run: /mdk/doc/manual/bin/checkForMissingHlp.sh
    • Check, if all HTML entities are in the HTML to LaTeX translation index. Run: /mdk/doc/manual/bin/checkForMissingHTML2LatexEntities.sh
    • Check, if there are all screenshots present. Run: /mdk/doc/manual/bin/checkMissingScreenshots.sh
    • Generate and update the screenshots
    • Run /mdk/bin/menuStart &rArr#Rightarrow;doc &rArr#Rightarrow;manual
    • Choose the language: &rArr#Rightarrow;lang. Choose all
    • Generate the tex files: &rArr#Rightarrow;text
    • Optimise the PNGs. &rArr#Rightarrow;optimisePNGs
    • Generate PDF and HTML version of the manual: &rArr#Rightarrow;pdf-html
    • Upload: &rArr#Rightarrow;upload
    • Go back
  • Development guide
    • Add (maybe existing new) LaTeX files
    • Build the development guide: &rArr#Rightarrow;devguide
    • Generate it: &rArr#Rightarrow;generate
    • Upload: &rArr#Rightarrow;upload
