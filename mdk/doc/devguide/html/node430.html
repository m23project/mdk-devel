CClient::setPackageProxy

Description: Sets the IP and port of the package proxy.
Parameter:
  • proxyIP: The IP of the package proxy.
  • proxyPort: The port of the package proxy.
Returns:
  • true, if the package IP and port are valid and set otherwise false.
