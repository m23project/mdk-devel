CFDiskIO::getBiggestLowestValueOf

Description: Gets the biggest or lowest value from all partitions of a given type.
Parameter:
  • vDisk: Virtual (internally used) disk number.
  • partType: Type of the partition (CFDiskIO::PT_PRIMARY, CFDiskIO::PT_EXTENDED, CFDiskIO::PT_LOGICAL, CFDiskIO::PT_EFI) or * for all types.
  • varType: Variable in the associative array ($this-wantedPartitioning[$vDisk][$vPart][$varType]) to check.
  • biggest: If set to true, the biggest value is searched, otherwise false.
Returns:
  • Biggest or lowest value from all partitions of a given type.
