AUTOTEST_sshTunnelOverServer

Description: Executes commands on a client indirectly, that is reachable over an m23 server only. SSH connections require root user on the m23 server and the client.
Parameter:
  • serverNameOrIP: The name of the m23 server or its IP.
  • clientNameOrIP: The name of the client or its IP.
  • cmds: The commands to execute on the client
  • password: Parameter to set the SSH password or NULL to use SSH keys only.
Returns:
  • The output of the client's screen.
