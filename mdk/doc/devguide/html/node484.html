CFDiskBasic::createSwapPartition

Description: Creates and formats a swap partition. The created partition is stored in the client parameters as swap partition.
Parameter:
  • diskDev: The disk device (e.g. /dev/hda) where the swap partition should be created on.
  • instStart: Start position of the swap partition (in MB).
  • instEnd: End position of the swap partition (in MB).
