CAutoUpdate::getAllGroupsHumanReadable

Description: Generate a list of all groups (with optional button for deleting an entry).
Parameter:
  • id: ID of the set to get all groups.
  • addButtons: If set to true, each group will get a deletion button.
  • defaultSeparator: The (HTML) string to separate the entries.
Returns:
  • List of all groups (with optional button for deleting an entry).
