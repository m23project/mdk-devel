UCS_getUDMInfo

Description: Get information from the UCS's LDAP by udm tool.
Parameter:
  • udmModule: Name of the udm module (eg. networks/network or computers/linux).
  • afterLines: Amount of lines with information after the line containing the keyword.
  • keyWord: Keyword matching the first line to find the block with the wanted information.
Returns:
  • : Associative array with the information.
