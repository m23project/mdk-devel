CFDiskIO::isFreeSpaceBetween

Description: Checks if there is a free space between $start and $end and the type of partition could be created there.
Parameter:
  • vDisk: Virtual (internally used) device number.
  • type: type of the partition (primary, extended, logical, efipart)
  • start: start position for the search
  • end: end position for the search
  • addErrorMsg: Set to true, if the error messages should be added.
Returns:
  • true, if the there is free space, otherwise false.
