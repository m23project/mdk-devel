CPool::isPackageTaskRunning

Description: Checks if a packages task (download to the pool or creating the pool from downloaded files) is running.
Parameter:
  • singleScreenName: Name of the single screen, that is used when downloading and creation of the pool are two seperate screen sessions.
Returns:
  • : true, if the task is running, otherwise false.
