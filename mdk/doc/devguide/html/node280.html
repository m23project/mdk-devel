CChecks::checkClientname

Description: Checks if a client name is valid (and optionally, if the client doesn't exist) and adds an error message to the message manager in case of an error.
Parameter:
  • clientName: Clientname to check.
  • checkNonused: Set to true, to check for non using.
Returns:
  • true, if the clientname is correct otherwise false.
