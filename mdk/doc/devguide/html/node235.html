CAutoUpdate::addStartEnd

Description: Adds a start end range for running the updates.
Parameter:
  • id: ID of the set to add the time range.
  • start: Start time of the range as combined numeric day and hour/minute string.
  • End: End time of the range as combined numeric day and hour/minute string.
Returns:
  • true, if the time range could be added, otherwise false.
