Usage 

partinfodb <dbuser> <dbpassword> <serverIP> <clientname> <cut character>
  • dbuser: user name for the database account
  • dbpassword: password for the database account
  • serverIP: ip of the database server
  • clientname: name of the client
  • cut character: Select '.' for english and ',' for german version. You can find out the language setting very easy: the messages printed in the console have the same language that will used for the splitter.
