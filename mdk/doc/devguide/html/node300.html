CClientLister::addStatusFilter

Description: Adds an OR filter to get only clients that gave the given status (stati).
Parameter:
  • operator: Operator (can be '=', '', '') selects if the client status should be equal, smaler or bigger that the given status.
  • status: Status to compare with the state of the client
