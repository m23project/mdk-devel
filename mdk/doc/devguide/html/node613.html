CFDiskIO::collidesWithPartitionOfType

Description: Checks if a partition (to be created) defined by start and end position would collide with existing partitions of given type.
Parameter:
  • vDisk: Virtual (internally used) device number.
  • type: type of the partition (primary, extended, logical, efipart, or -1 to match all partition types)
  • start: start position for the partition (to be created)
  • end: end position for the partition (to be created)
Returns:
  • true, if the partition to create would collide, otherwise false.
