CObjectStorageManager::getCOSI18NVariable

Description: Gets the contents of an I18N variable for the object (if it exists).
Parameter:
  • object: Object to use or null, if $this-object should be used.
  • varPostfix: String to add to the name of the I18N variable.
  • errMsg: Die message to display if the I18N variable is not set.
Returns:
  • Contents of the the I18N variable.
