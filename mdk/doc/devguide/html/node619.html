CFDiskIO::nextPrimaryDevNr

Description: Gets the next free EFI (physical) partition number.
Parameter:
  • vDisk: Virtual (internally used) disk number.
Returns:
  • Next free EFI (physical) partition number or -1, if there are no free partition numbers.
