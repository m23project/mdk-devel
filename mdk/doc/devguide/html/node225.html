CAutoUpdate::getUpdateTypeTranslation

Description: Returns the whole translation array with update type as key and the translation as value or the translation for a given constant.
Parameter:
  • const: If set to a constant, the translation will be returned, otherwise the whole array.
Returns:
  • : Translation array, translation for a given constant or false, if the constant is not valid.
