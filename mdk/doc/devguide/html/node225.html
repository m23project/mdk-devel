CAutoUpdate::getAllClientnames

Description: Gets the clients in all groups of the active set.
Parameter:
  • running: If set to true, only clients running auto update will be listed, otherwise all clients.
Returns:
  • Array with the clients in all groups of the active set or empty array in case of an error.
