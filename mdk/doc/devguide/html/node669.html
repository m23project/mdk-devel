CFDiskTest::deletePartition

Description: Deletes a partition from a disk and checks for consistancy of the partioning between virtual calculation and actual client partitioning.
Parameter:
  • dev: The partition device (e.g. /dev/sda1).
  • deleteBelongingRaid: If set to true, the RAID, the partition belongs to will be destroyed.
