CFDiskIO::devNrExists

Description: Checks if a certain (physical) partition number exists.
Parameter:
  • vDisk: Virtual (internally used) device number.
  • devNr: Device numer you want to check (e.g. 2 for /dev/sda2)
Returns:
  • true, when the (physical) partition number exists on the disk.
