CLCFG_mountRootDir

Description: create a new directory for mounting the root partition of the client, mount the installation directory to this mount point and create a temp directory
Parameter:
  • rootDev: the root device (e.g. /dev/hda1)
  • mountPoint: directory under /mnt to mount the device in
  • CFDiskIOO: Client object (used for mounting the EFI boot partition).
