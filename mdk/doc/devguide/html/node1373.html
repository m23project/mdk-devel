DISTR_listCommaSeperated

Description: returns a normal array with the values of the specified variable (e.g. var: GUIs = result: [0] = Textmode [1] = KDE3 [2] = KDEwoody ...)
Parameter:
  • variable: the variable to search for
  • values: the values of the distribution text file (info.txt) read with DISTR_getDescriptionValues
