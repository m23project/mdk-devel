CFDiskGUI::addDiskLayoutEntry

Description: Adds a new entry (partition or free space on a disk) to the disk layout (for rendering only).
Parameter:
  • vDisk: Internal disk number (for accessing the disk information in the array).
  • start: start point of the partition (in MB)
  • end: end point of the partition (in MB)
  • type: type of the partition (primary, logical)
  • fs: File system.
  • dev: Device name (optional)
