CFDiskBasic::getMknodCommand

Description: Generates the mknod command for a given /dev/sdX(Y) device (disk or partition).
Parameter:
  • dev: The device (e.g. /dev/sda5) to created the mknod command for.
Returns:
  • mknod command with the parameter matching the given /dev/sdX(Y).
