AUTOTEST_SELENIUM_exec

Description: Runs a command on the HTTP2SeleniumBridge.
Parameter:
  • bridgeURL: URL to the HTTP2SeleniumBridge server (eg. http://192.168.1.153:23080)
  • cmd: The command to execute (eg. typeInto)
  • ID or name: ID or name of the HTML element to
  • additional parameters: Additional parameters with variable name first, value second.
Returns:
  • HTTP output of the HTTP2SeleniumBridge call.
