The files

All files for an update have to begin with the version number (e.g. 0.4.5):
  • 0.4.5.begin: conatins the bas script that is executed a the beginning
  • 0.4.5.end: conatins the bas script that is executed a the end
  • 0.4.5.info: contains only the codename of the m23 release (e.g. shiver)
  • 0.4.5.text: helds the decribing text of the update. There you should put a changelog
