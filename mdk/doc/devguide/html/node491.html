CFDiskBasic::deleteDeviceFromRaid

Description: Deletes a partition or disk from a RAID disk.
Parameter:
  • raidDev: Device name of the RAID (eg. /dev/md0)
  • devToRemove: Partition or disk device name to remove.
Returns:
  • true, if the device could be removed, otherwise false.
