CPool::createBasicPool

Description: Sets the name, type and architecture of the pool and creates the pool directory.
Parameter:
  • poolName: Name of the pool.
  • poolType: Type of the pool (POOL_TYPE_CD or CPoolLister::POOL_TYPE_DOWNLOAD).
  • poolArch: Architecture of the pool (POOL_ARCH_I386 or CPoolLister::POOL_ARCH_AMD64).
