CChecks::firstpw

Description: Checks if the first name's password is valid.
Parameter:
  • pass: The password to check.
  • allowEmpty: Set to true, if empty passwords should be allowed (e.g. when read from an LDAP server)
Returns:
  • true, if the password is valid otherwise false.
