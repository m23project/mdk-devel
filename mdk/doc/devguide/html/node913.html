CPackageStatusCompare::setFile

Description: Sets a file name, if the given parameter is a valid file name.
Parameter:
  • var: Variable that stores a client or file name.
  • file: Name of the status file.
  • origVar: Variable that stores the original file name.
  • origFile: Original name of the status file.
