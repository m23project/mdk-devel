GRP_showSelDistrSources

Description: shows a dialog for selection of distribution and package source name. The choices are taken form distr and packagesource values of the clients in the group. If there is only one entry for one or both of the values, the value is written back to the input variable otherwise a HTML selection is shown.
Parameter:
  • groupNames: group names stored in an array or NULL, if groups should be ignored and all distributions and sources lists should be shown.
  • distr: distribution to show first and variable to write the distribution name back
  • sourceslist: sources list to show first and variable to write the sources list name back
