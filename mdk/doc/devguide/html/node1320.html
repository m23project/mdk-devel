CLCFG_dialogGaugeProcPos

Description: Generates the BASH code for showing a dialog with status bar that uses /proc/pid/fdinfo for current file position
Parameter:
  • backtitle: title on top of the screen
  • title: message title
  • message: message to show over the status bar
  • infofilecmd: BASH code to figure out the needed status file in the proc filesystem
  • fullsize: The full size of the file that should be written.
  • force: Forces showing of the dialog box
