CChecks::checkEmail

Description: Checks if the given eMail is valid (or optionally empty).
Parameter:
  • email: eMail address to check.
  • allowEmpty: Set to true, if empty eMail addresses should accepted as valid.
Returns:
  • true, if the eMail address is valid (or empty) otherwise false.
