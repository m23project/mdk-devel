CFDiskBasic::createUEFIPartition

Description: Creates a new UEFI partition with a size of 512 MB at the start of the disk (if possible).
Parameter:
  • dev: The device (e.g. /dev/sda).
Returns:
  • The physical partition number of the newly created partition.
