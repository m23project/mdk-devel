\section{Ajuster un poste client}Vous voyez une liste des postes client dont la d\'et\'ection du mat\'eriel est termin\'ee. Maintenant, vous avez la possibilit\'e de les partitionner et formater.\\
\includegraphics[scale=0.4]{/mdk/doc/manual/screenshots/fr/clients_setup.png} \\
Pour pouvoir partitionner et formater le client, cliquez sur \textit{$\ll$Configuration$\gg$}.\\
Quand vous cliquez sur le nom d'un client, vous recevrez des informations détaillées sur le client sélectionné et vous auriez accès au centre de contrôle.\\
\subsection{Signification des couleurs symboliques}
Le couleur du symbol d'état indique l'état de l'installation du poste client.\\
\begin{itemize}
\item \textbf{rouge}: Le poste client a été ajouté et la détection du matériel automatique n'est pas encore terminée.\\
\item \textbf{jaune}: Vous pouvez formater et partitionner le poste client, le système de base sera assigné automatiquement.\\
\item \textbf{vert}: Le système de base du poste client est établi et vous pouvez installer du logiciel supplémentaire.\\
\item \textbf{bleu}: Le poste client est en train d'installer du logiciel supplémentaire.\\
\item \textbf{orange}: Le poste client est dans un \textbf{état critique}. Cela veut dire qu'il y a eu une erreur pendant l'installation qui doit être écartée par l'administrateur. Au centre de contrôle, il y a des possibilités différentes d'écarter l'état critique.\\
\item \textbf{blanc}: Poste client modèle pour l'installation de masse dont les configurations seront transmis aux autres postes client.\\
\item \textbf{Mouche (bogue)}: Indique que le poste client se trouve en état de débogage.\\
\end{itemize}
\subsection{Travaux}
Dans cette colonne, vous trouvez le nombre des tâches pas encore acomplies devant et le nombre de toutes les tâches après la barre oblique.\\
\subsection{Travailler plusieurs postes clients}
Vous pouvez choisir plusieurs postes client en mettant un crochet dans la colonne avec les postes clients souhaités.\\
Puis, vous pouvez exercer des actions différentes avec les postes client choisis:\\
\begin{itemize}
	\item \textbf{Effacer d'un groupe}: Sélectionnez le groupe duquel vous voulez effacer les postes client. Si un poste client n'appartient pas à ce groupe, aucune action sera excercé avec lui.\\
	\item \textbf{Ajouter à un groupe}: Sélectionnez le groupe auquel vous voulez ajouter les postes client. Si un poste client appartient déjà à ce groupe, rien ne sera changé à sa dépendance.\\
	\item \textbf{Effacer}: Efface les postes client sélectionnés\\
\end{itemize}
\subsection{Notez}
Pour voir l'état actuel de vos postes client, utilisez la fonction de rechargement de votre navigateur web (par exemple en appuyant sur la touche F5)\\
\subsection{Astuces}
En cliquant sur le symbole d'état d'un poste client, vous pouvez changer l'état actuel du poste client. Quand même, vous devriez seulement changer l'état d'un poste client si vous savez exactement ce que vous faites.\\
Le mode de débogage peut être (dés)activé par un clic sur le symbole avec la mouche.\\
