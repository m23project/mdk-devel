\section{Client-Direkt-Verbindung}Eine Client-Direkt-Verbindung kann dazu verwendet werden, um manuell in den Installations-Prozeß eines Clients einzugreifen, oder die Bildschirmausgabe von einem entfernten Ort aus zu lesen, sowie zu Test- und Debug-Zwecken. Die Verbindung wird per SSH aufgebaut und kann so mit jedem SSH-Client hergestellt werden.\\
\includegraphics[scale=0.4]{/mdk/doc/manual/screenshots/de/client_directConnection.png} \\
\subsection{Paßwort}
Ist der Client noch in einem frühen Installationsstadium, so lautet das Root-Paßwort: "NETROOTPWD"\\
Später ist es das Paßwort, das Sie beim Anlegen des Clients vergeben haben.\\
Sollte Ihr Browser das SSH-Protokoll direkt unterstützen, so können Sie die Verbindung durch Anklicken dieses Links herstellen: ssh://root@CLIENTIP\\
Sonst öffnen Sie bitte eine Konsole und geben den Befehl: \textbf{ssh -o UserKnownHostsFile=/dev/null -l root CLIENTIP}
Um in die aktuelle Client-Bildschirmausgabe zu lesen oder in die laufende Installation einzugreifen benutzen Sie den Befehl: \textbf{screen -x m23install}
