\section{Erweiterte Partitionierung: Partitionen anlegen}
\includegraphics[scale=0.4]{/mdk/doc/manual/screenshots/de/fdisk-extended1.png} \\
\begin{itemize}
\item Nun können Sie neue Partitionen anlegen. Wählen Sie aus, ob Sie eine primäre (primary), erweiterte (extended) oder logische (logical) Partition erstellen möchten.\\
Wählen Sie den Typ und den Start- und End-Punkt der Partition und klicken Sie danach auf \textit{"Hinzufügen"}. Unter Typ können Sie nur Partitionstypen auswählen, die Sie tatsächlich anlegen können. Zu den Bedingungen zum Anlegen der Partitionstypen finden Sie unten einen Hinweis.\\
\item Unter \textit{"Partitions-Überblick"} finden Sie freie Bereiche (weiß gekennzeichnet), in denen Sie neue Partitionen anlegen können. Als Anhaltspunkt für die Start- und End-Werte Ihrer neuen Partitionen beachten Sie bitte die Werte unter \textit{"Bereich"}.\\
\item Sollte nicht genügend freier Platz vorhanden sein, um die von Ihnen gewünschten Partitionen anzulegen, so klicken Sie wiederholt auf den \textit{"Zurück"}-Button Ihres Browsers, bis Sie zur \textit{"Partitionen löschen"}-Seite gelangen.\\
\item Haben Sie alle benötigten Partitionen angelegt, dann klicken Sie auf \textit{""}.\\
\end{itemize}
\subsection{Information: Partitionstypen}
\begin{itemize}
 \item \textbf{primär:} Sie können maximal 4 primäre Partitionen anlegen.\\
 \item \textbf{erweitert:} Eine erweiterte Partition zählt ebenfalls zu den primären Partitionen. Es kann maximal eine erweiterte Partition angelegt werden. Innerhalb einer erweiterten Partition können Sie logische Partitionen anlegen, falls Ihnen 4 primäre nicht reichen sollten.\\
 \item \textbf{logisch:} Logische Partitionen können nur im Speicherplatz einer erweiterten Partition angelegt werden. Die Anzahl der logischen Partitionen ist nicht beschränkt.\\
\end{itemize}
