\section{Erweiterte Partitionierung: Partitionen löschen}
\includegraphics[scale=0.4]{/mdk/doc/manual/screenshots/de/fdisk-extended0.png} \\
Wählen Sie die Partition, die Sie löschen möchten und klicken Sie danach auf \textit{"Löschen"}. Gehen Sie nach dem Löschen zum nächsten Schritt weiter, indem Sie auf \textit{""} klicken.\\
\subsection{Hinweis}
Möchten Sie keine Partitionen löschen, so gehen Sie gleich zum nächsten Schritt über.\\
