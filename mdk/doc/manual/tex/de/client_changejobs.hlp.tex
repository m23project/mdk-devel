\section{Aufträge ändern}
\includegraphics[scale=0.4]{/mdk/doc/manual/screenshots/de/client_changejobs.png} \\
Hier können Sie bereits abgearbeitete und wartende Aufträge des Clients begutachten und ggf. ändern. Es stehen Ihnen folgende Änderungsmöglichkeiten zur Verfügung:\\
\begin{itemize}
	\item Wählen Sie bei \textit{"Aktion"} \textbf{Löschen} aus, um den Auftrag endgültig zu verwerfen.\\
	\item \textbf{Wiederholen} setzt einen bereits abgearbeiteten Auftrag wieder auf die Liste der auszuführenden Aufträge.\\
	\item \textbf{Fertig} markiert einen Auftrag als abgeschlossen. Dieser wird deshalb nicht mehr ausgeführt.\\
\end{itemize}
Klicken Sie anschließend auf \textit{"Änderungen übernehmen"}, um die Änderungen durchzuführen. Möchten Sie zusätzlich, daß die von Ihnen auf \textbf{Wiederholen} gesetzte Aufträge sofort ausgeführt werden, so klicken Sie auf \textit{"Aufträge auf dem Client ausführen"}.\\
