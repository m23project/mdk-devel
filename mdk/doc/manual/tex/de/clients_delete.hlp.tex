\section{Client löschen}
\includegraphics[scale=0.4]{/mdk/doc/manual/screenshots/de/clients_delete.png} \\
Um einen Client zu löschen, klicken Sie auf \textit{"Löschen"}. Im folgenden Dialog können Sie sich noch vergewissern, daß es sich um den richtigen Client handelt.\\
Klicken Sie auf den Clientnamen, erhalten Sie detaillierte Informationen zu dem gewählten Client und haben Zugriff auf das Kontrollzentrum.\\
\subsection{Statuserklärung}
Die Farbe des Statussymbols gibt an, wie weit die Installation des jeweiligen Clients vorangeschritten ist.\\
\begin{itemize}
\item \textbf{rot}: Client wurde aufgenommen, die automatische Hardwareerkennung ist nicht abgeschlossen.\\
\item \textbf{gelb}: Sie können den Client formatieren und partitionieren, das Basissystem wird automatisch zugewiesen.\\
\item \textbf{grün}: Das Basissystem des Clients ist eingerichtet, Sie können zusätzliche Software installieren.\\
\item \textbf{blau}: Der Client installiert gerade zusätzliche Software.\\
\item \textbf{orange}: Der Client ist im \textbf{kritischen Zustand}. D.h. bei der Installation ist ein Fehler aufgetreten, der vom Administrator behoben werden muß. Im Kontrollzentrum gibt es eine Reihe von Möglichkeiten, den kritischen Zustand zu beheben.\\
\item \textbf{weiß}: Musterclient für die Masseninstallation, von dem die Einstellungen auf die anderen Clients übertragen werden.\\
\item \textbf{Fliege (Bug)}: Zeigt an, daß sich ein Client im Debug-Modus befindet.\\
\end{itemize}
\subsection{Aufträge}
In dieser Spalte steht vor dem Schrägstrich die Anzahl der noch nicht abgearbeiteten Aufträge und die Anzahl aller Aufträge dahinter.\\
\subsection{Mehrere Clients bearbeiten}
Wählen Sie mehrere Clients dadurch aus, daß Sie den Haken in den Spalte mit den gewünschten Clients setzen.\\
Danach können Sie verschiedene Aktionen mit allen gewählten Clients durchführen:\\
\begin{itemize}
	\item \textbf{Aus Gruppe löschen}: Wählen Sie hierzu die Gruppe aus, aus der die Clients gelöscht werden sollen. Sollte ein Client nicht in der angegebenen Gruppe sein, so wird mit diesem keine Aktion durchgeführt.\\
	\item \textbf{Zur Gruppe hinzufügen}: Wählen Sie hierzu die Gruppe aus, zu der die Clients hinzugefügt werden sollen. Ist ein Client bereits in dieser Gruppe vorhanden, so wird an seiner Gruppenzugehörigkeit nichts geändert.\\
	\item \textbf{Löschen}: Löscht die gewählten Clients\\
\end{itemize}
\subsection{Hinweis}
Um den aktuellen Client-Status zu erhalten, benutzen Sie die Reload-Funktion Ihres Browsers (z.B. durch Drücken der Taste F5)\\
\subsection{Tricks}
\begin{itemize}
\item Durch Klicken auf das Statussysmbol kann der aktuelle Status des Clients geändert werden.\\
\item Allerdings sollten Sie den Client-Status nur ändern, wenn Sie wissen, was Sie tun. Der Debug-Modus kann durch einen Klick auf das Laus-Icon (de)aktiviert werden.\\
\end{itemize}
