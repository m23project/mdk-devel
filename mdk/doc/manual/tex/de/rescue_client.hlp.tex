\section{Notfallsystem starten}Möchten Sie zu Reparatur- oder Diagnosezwecken einen Client booten, so können Sie das Notfallsystem über Netzwerk booten. Nach dem Booten startet der Client eine Konsole, auf der Sie arbeiten können. Klicken Sie zum Starten des Notfallsystems auf \textit{"Notfallsystem"} hinter dem Clientnamen.\\
\includegraphics[scale=0.4]{/mdk/doc/manual/screenshots/de/rescue_client.png} \\
\subsection{Hinweis}
Das Notfallsystem wird solange nach dem Booten gestartet, bis Sie den Job \textit{"m23Rescue"} aus der Auftragsliste löschen. Diese finden Sie, indem Sie auf der Seite \textit{"Clients-Überblick"} auf die Zahl in der Spalte \textit{"Aufträge"} klicken.\\
