\section{Backup}
\includegraphics[scale=0.4]{/mdk/doc/manual/screenshots/de/client_backup.png} \\
In diesem Dialog können Sie alle Verzeichnisse des Clients angeben, von denen ein Backup erstellt werden soll. Geben Sie diese Verzeichnisse dazu durch Kommata getrennt in der Eingabezeile bei \textit{"Backup-Verzeichnisse"} an und klicken Sie anschließend auf \textit{"Speichern"}.\\
Das eigentliche Backup übernimmt das Programm BackupPC, das Sie mit einem Klick auf das Icon starten können. In der BackupPC-Oberfläche haben Sie Zugriff auf zusätzliche Backup-Funktionen.\\
