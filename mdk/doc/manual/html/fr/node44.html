scp

Pour ''Secure Copy'' les indications suivantes sont nécessaires:
  • Utilisateur: C'est le nom d'utilisateur avec lequel le serveur m23 doit se connecter sur le serveur SCP pour y sauvegarder le fichiers de sauvegarde.
  • Serveur SCP: Ici, entrez le nom ou l'adresse IP par le(la)quel(le) le serveur m23 se connectera au serveur SCP.
  • Répertoire de sauvegarde du serveur: C'est le répertoire sur le serveur SCP où les archives de sauvegarde seront copiés. Entrez le chemin absolu.
  • Clé GPG: Ici, choisissez la clé GPG avec laquelle vous souhaitez chiffrer les sauvegardes avant de les transférer.
