Procédez comme suit:

  1. Entrez votre délimiteur (signe séparateur) dans le champ correspondant, puis adaptez le nombre de colonnes si nécessaire.
  2. Cliquez sur Actualiser La première ligne du fichier de données est maintenant délimité par le séparateur et affiché dans le champ des options du client.
  3. Assignez les colonnes du fichier de données ainsi séparées aux propriétés de vos postes client.
  4. Enregistrez votre sélection en cliquant sur Enregistrer.
