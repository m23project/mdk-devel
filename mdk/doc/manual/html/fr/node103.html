Information supplémentaire:

Après ajout du nouveau client, ce dernier démarrera sur le réseau (netboot) pour effectuer une séquence de détection du matériel, durant laquelle les informations collectées, comme par ex. la taille du disque dur utilisé, seront envoyées au serveur.
Après l'achèvement du transfert des données l'état du client passera à jaune vous pourrez dès lors ajuster les paramètres du client. Pour cela, cliquez sur Postes client puis sur Configuration dans le menu.
