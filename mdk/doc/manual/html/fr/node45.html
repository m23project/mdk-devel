Remarque

Prenez soin que la clé SSH du superutilisateur root du serveur m23 qui est contenu dans le fichier 
/root/.ssh/id\_dsa.pub
serait entré dans le fichier 
~/.ssh/authorized\_keys
de l'utilisateur indiqué Utilisateur sur le serveur SCP pour qu'un transfert des fichiers sans mot de passe soit possible. Si vous disposez de la commande ssh-copy-id, la copie s'effectuera simplement. Pour ce faire, tapez la commande ssh-copy-id -i /root/.ssh/id_dsa.pub user@serveur_SCP.
