Signification des couleurs symboliques

Le couleur du symbol d'état indique l'état de l'installation du poste client.
  • rouge: Le poste client a été ajouté et la détection du matériel automatique n'est pas encore terminée.
  • jaune: Vous pouvez formater et partitionner le poste client, le système de base sera assigné automatiquement.
  • vert: Le système de base du poste client est établi et vous pouvez installer les logiciels supplémentaires.
  • bleu: Le poste client est en train d'installer des logiciels supplémentaires.
  • orange: Le poste client est dans un état critique. Cela veut dire qu'une erreur s'est produite pendant l'installation; elle doit être inhibée par l'administrateur. Dans le centre de contrôle, plusieures possibilités d'inhiber l'état critique sont offertes.
  • blanc: Poste client modèle pour l'installation de masse dont les configurations seront transmises aux autres postes client.
  • Mouche (bogue): Indique que le poste client se trouve en état de débogage.
