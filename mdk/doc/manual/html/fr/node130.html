Information pour l'installation d'un système d'exploitation sur des systèmes RAID

Lorsque le système d'exploitation doit être installé sur un lecteur RAID, il est nécessaire dans la plupart des cas (sauf pour RAID-1) que vous établissiez une partition supplémentaire pour le noyau et les modules (/boot), ne faisant pas partie du lecteur RAID. Une taille d'environ 250 MB suffit à cette partition pour conserver plusieures versions de noyau. S'il n'y avait plus de partition libre, cliquez sur le bouton Précedent jusqu'à ce que vous atteigniez la création et l'effacement de partitions puis créez la partition nécessaire (/boot) avant le lecteur RAID.
Partition pour l&apos;installation et le swap Pour l'installation du poste client m23, vous devez choisir une partition pour l'installation du système d'exploitation sous Partition pour l&apos;installation et une partition pour l'espace de swap sous Partition pour le swap, puis cliquez sur Choisir.
Définir les points de montage Si vous désirez des points de montage additionels, vous pouvez les définir sous Définir les points de montage. Entrez la partition, le point de montage et les paramètres requis au montage, puis cliquez sur Ajouter. Ces informations correspondent à celles pouvant être trouvées dans le fichier /etc/fstab d'un système Linux. Vous pouvez voir les points de montage déjà définis dans la sous le champs de saisie. Si vous désirez effacer un point de montage créé, cliquez sur Effacer sur la ligne concernée.
