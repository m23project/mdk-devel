Opérer sur plusieurs postes clients

Sélectionnez plusieurs postes client en les cochant dans la liste correspondante.
Vous pouvez lancer les actions sur les postes client choisis:
  • Effacer d&apos;un groupe: Sélectionnez le groupe duquel les postes client seront à retirer. Si un poste client n'appartient pas dans ce groupe, aucune action ne sera excercée sur lui.
  • Ajouter à un groupe: Sélectionnez le groupe auquel seront ajoutés les postes client. Si un poste client est déjà dans ce groupe, son appartenance au groupe restera inchangée.
  • Effacer: Efface les postes client sélectionnés
  • Déplacer dans ce groupe: Cette fonction déplace un poste client dans un groupe différent.
  • Restaurer: Réinstallation complète des postes clients.
  • Re-exécuter les tâches du poste client: Répète toutes les tâches ayant déja été exécutées sur le client m23.
  • Comparer l'état de paquets: Comparaison de l'état des paquets entre deux postes clients.
