Intégration manuelle

Si aucun démon SSH n'est actif sur le poste client, vous avez également la possibilité de lancer le processus d'intégration manuellement. Pour ce faire, exécutez les commandes suivantes en tant que root ou équivalent (sudo) dans la console du poste client (remplacez serverIP par l'adresse IP de votre serveur m23): 
cd /tmp; wget http://$\langle$serverIP$\rangle$/work.php -O work.php; sh work.php
