Error handling

Si votre matériel ne peut être assimilé, vous pouvez utiliser l'''Assimilation Manuelle''. Vous devrez ajouter l'ID du client (Le numéro entre parenthèses suivant le nom du client dans ''Vue d&apos;ensemble des postes clients''). La commande complète ressemble maintenant à cela: 
cd /tmp; wget http://$\langle$serverIP$\rangle$/work.php?m23clientID=$\langle$clientID&gt -O work.php; sh work.php
