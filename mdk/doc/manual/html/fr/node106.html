Les fichiers image

Les fichiers seront enregistrés dans le répertoire /m23/data+scripts/clientImages
sous différents types et compressés dans des formats distincts.
Les noms de fichiers sont toujours composés selon le même principe: nom de l'imagetaille de l'image decompressée en bytesformat de l'imagecompression
L'image sera créée comme suit:
Pour la commande:
  • dd: Toutes les données d'une partition ou d'un lecteur seront enregistrées (block par block).
Pour la compression, les valeurs suivantes sont valides:
  • (aucune extension de compression): Le fichier image n'est pas comprimé.
  • gz: La compression sera exécutée avec le programme gzip.
  • bz2: La compression sera exécutée avec le programme bzip2 (le plus efficient dans la plupart des cas).
