Choisir le fichier de données et le télécharger

Choisissez un fichier par le dialogue, puis, cliquez sur Téléverser.
Un format spécifique pour les données est requis afin que m23 puisse identifier les valeurs. Les valeurs des propriétés d'un poste client sont spécifiées sur une seule ligne. Chaque propriété est séparée des autres par un signe séparateur (ici: ;): 
propri\'et\'e1;propri\'et\'e2;propri\'et\'e3;...
A la place du point-virgule, vous pouvez utiliser n'importe quelle combinaison d'un maximum de 4 lettres/chiffres. L'ordre des propriétés doit être le même dans chaque ligne.
