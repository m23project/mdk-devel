Lancer le système d'urgence

Pour démarrer un poste client dans un but de réparations ou de diagnostiques, vous pouvez démarrer le système d'urgence par le réseau (netboot). Après le démarrage, le client lance une console sur laquelle vous pouvez travailler. Pour lancer le système d'urgence, cliquez sur Système d&apos;urgence derrière le nom du client.
