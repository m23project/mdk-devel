Renseignement concernant l'architecture du processeur

m23 est compatible avec les processeurs 32 et 64 bits. Une version 32 bits peut être installée sur un ordinateur 64 bits, l'inverse n'est pas possible. Pour profiter de l'entière potentialité d'un microprocesseur 64 bits, vous devez choisir amd64. Cette catégorie amd64 ne comportent pas uniquement les processeurs AMD, mais également ceux d'Intel. Par exemple, les processeurs Intel Pentium D, Pentium Extreme Edition, Celeron D, Core 2 et les processeurs AMD Sempron, Athlon 64, Athlon X2 et Phenom en font partie.
