Chercher et remplacer

  • Chercher: Ctrl + F (PC), Command + F (Mac)
  • Chercher la prochaine occurence: Ctrl + K (PC), Command + G (Mac)
  • Chercher la précédente occurence: Ctrl + + K (PC), Command + + G (Mac)
  • Chercher et remplacer: Ctrl + H (PC), Command + Option + F (Mac)
  • Sauter à la ligne numéro: Strg + L (PC), Command + L (Mac)
