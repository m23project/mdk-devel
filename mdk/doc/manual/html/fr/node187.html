Éditeur de scripts


Les scripts simplifient l'exécution de tâches répétitives sur vos postes clients en la rendant aussi facile que l'installation d'un paquet. En utilisant l'éditeur, vous pouvez créer vos propres scripts et profiter de toute l'étendue possible de l'environnement fourni par m23, pour générer le code BASH afin que ces scripts puissent être exécuté sur les postes clients. Vous pouvez utiliser toutes les commandes BASH et tous les programmes installés sur les postes clients. L'index des commandes de m23 est consultable dans le Development Guide, dans le menu de gauche.
  • Ouvrir des scripts existants: Sélectionnez le script souhaitée dans la liste sous Gérer les scripts puis, cliquez sur Charger. Ensuite, le texte apparait dans la fenêtre de l'éditeur.
  • Enregistrer et effacer des scripts: Cliquez sur Enregistrer ou sur Effacer. Les actions se réfèrent au script qui est actuellement chargée.
  • Sauvegardez le script sous un nom différent: Si vous souhaitez sauvegarder le script dans l'éditeur sous un nouveau nom, saisisez ce nouveau nom dans ''Sauvegarder en tant que'' puis, cliquez sur ''Enregistrer''.
  • Création d'un nouveau script: Entrez un nom pour le nouveau script par Créer une nouveau script et confirmez-le en cliquant sur Créer. Un exemple de script court, que vous pouvez adapter à vos besoins, sera affichée dans la fenêtre de l'éditeur.
  • Télécharger des scripts appartenant à d'autres utilisateurs m23: Vous pouvez charger les scripts publiés par d'autres utilisateurs directement depuis l'éditeur. Pour ce faire, mettez à jour la liste des scripts disponibles en cliquant sur ''Modifier la liste des scripts en ligne''. Sélectionnez ensuite le script désiré en cliquant sur ''Télécharger le scripts'' puis chargez le dans l'éditeur en cliquant sur ''Charger''
  • Partagez vos scripts avec les autres utilisateurs de m23: L'éditeur vous offre une possibilité simple de partager vos scripts avec des autres utilisateurs de m23 et par conséquent, d'en faire profiter un plus grande communauté d'utilisateurs. Tous ces scripts seront publiées sur le site web de m23 dans la rubrique Scripts of m23 users. Pour la publication de votre script, qui est actuellement chargée dans l'éditeur, entrez une description de la fonction du script sous Description. En outre, entrez votre code d'identification, nom ou un autre mot d'identification sous Auteur. Ensuite, cliquez sur Publier le script sur m23.sf.net. Votre script sera immédiatement chargé et apparaîtra sur le site web de m23.
  • Examinez la sortie du script: Pour le premier test de votre script, vous pouvez vérifier ses résultats, afin par exemple, de corriger les éventuelles erreurs de syntaxe. Pour ce faire, le script sera temporairement assigné à un client, et le Code-BASH résultant affichera ce qui aurait du aparaitre dans la fenêtre du client. Sélectionnez le client dans la liste ''Nom du poste client'' ci-dessous ''Visualiser le résultat du script''puis cliquez sur ''Ouvrir la visualisation du résultat du script''.
