Indication concernant l'utilisation de JavaScript

L'activation de JavaScript est optionnelle. Lorsqu'il est activé, quelques fonctions additionnelles de comfort seront à votre disposition (p.ex. l'ouverture automatique de l'index du menu ou le marquage de la partition se trouvant sous le pointeur de la souris). Néanmoins, toutes les fonctions nécessaires sont aussi disponibles sans JavaScript.
