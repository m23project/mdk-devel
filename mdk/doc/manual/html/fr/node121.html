Alternativement: Création d'un média d'amorçage

Lorsque la modification de la configuration de votre serveur DHCP n'est pas possible, vous pouvez toujours démarrer votre client m23 à partir d'une clé USB ou d'un CD/DVD, puis depuis le shell du client, exécuter le script '' 
fetchJobManual
'' pour démarrer le processus d'installation du client. De plus amples informations concernant la création du média d'amorçage sont disponibles sur ''Création d'un média d'amorçage'' dans ce menu.
