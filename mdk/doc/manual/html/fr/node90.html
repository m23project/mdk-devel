Travailler plusieurs postes clients

Vous pouvez choisir plusieurs postes client en mettant une coche dans la colonne des postes clients souhaités.
Puis, vous pouvez exercer des actions différentes sur les postes client choisis:
  • Effacer d&apos;un groupe: Sélectionnez le groupe duquel vous voulez effacer les postes client. Si un poste client n'appartient pas à ce groupe, aucune action sera excercé sur lui.
  • Ajouter à un groupe: Sélectionnez le groupe auquel vous souhaitez ajouter les postes client. Si un poste client appartient déjà à ce groupe, sa dépendance restera inchangée.
  • Déplacer dans ce groupe: Cette fonction déplace un poste client dans un groupe différent.
  • Restaurer: Réinstallation complète du poste client.
  • Effacer: Efface les postes client sélectionnés
