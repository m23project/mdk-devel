Mot de passe

Si le poste client est encore au début de l'installation, le mot de passe root est: NETROOTPWD.
Plus tard, ce sera le mot de passe saisi pendant la création du poste client.
Si votre navigateur web supporte le protocole SSH directement, vous pouvez établir la connexion en cliquant sur ce lien: ssh://root@CLIENTIP
Sinon, ouvrez un terminal et entrez la commande: ssh -o UserKnownHostsFile=/dev/null -l root CLIENTIP.
Afin de lire l'affichage actuel du poste client ou d'intervenir sur l'installation en cours utilisez cette commande: screen -x m23install.
