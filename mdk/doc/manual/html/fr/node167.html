Installer des paquets:

  1. Choisissez Combinaisons de paquets, Paquets normaux ou Paquets spéciaux (pour utilisateurs expérimentés).
  2. Entrez le critère de recherche pour le paquet dans le champ de saisie Recherche des paquets ou laissez vide ce champs pour chercher tous les paquets d'un type donné. A cause de la multitude de paquets existant, cette option n'est pas possible pour les paquets standards - dans ce cas, il est obligatoire d'entrer un critère de recherche.
  3. Sous Paquets trouvés, s'affichent tous les paquets correspondant à vos critères de recherche.
  4. Cliquez sur la case à cocher derrière les paquets souhaités.
  5. Notez les paquets pour l'installation en cliquant sur Noter.
  6. Répétez les étapes 1 à 5 selon vos besoins.
Maintenant, vous avez préselectionné des paquets pour l'installation. Une coche verte indique que ce paquet est à installer, une croix rouge indique qu'il est à effacer du client. Vous pouvez effacer la liste de ces paquets en cliquant sur Tout Annuler ou sélectionner des paquets (en cliquant sur les symboles en haut de la colonne avec les cases à cocher, vous pouvez sélectionner tous les paquets ou aucun paquet ou inverser votre sélection) et les effacer après en cliquant sur Annuler la sélection.
Confirmez votre choix en cliquant sur Installer les paquets sélectionnés.
