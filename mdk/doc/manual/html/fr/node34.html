Notez

Le CD d'installation du serveur m23 contient un script qui met à jour la version de m23 installée sur votre serveur. Procédez comme décrit ci-après pour mettre à jour m23:
  1. Ouvrez une console
  2. Insérez le CD d'installation du serveur de m23
  3. Montez le CD (par ex.: 
    mount /mnt/cdrom1
    )
  4. Allez sur le point de montage du CD (par ex.: 
    cd /mnt/cdrom1
    )
  5. Exécutez la mise à jour en entrant: 
    bash ./updatem23.sh
