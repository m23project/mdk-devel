Trois méthodes sont possibles:

  • Entrer: Les propriétés sont renseignées manuellement ou lues depuis un fichier.
  • Générer: Les valeurs seront générées automatiquement. Par exemple, les noms des postes client peuvent être générés avec des numéros incrémentaux ou avec des adresses IP inutilisées.
  • Garder: La valeur déterminée pour le poste client modèle sera utilisée pour tous les postes client.
    Pour certaines propriétés, votre choix est limitée. Les méthodes ne pouvant pas être choisies sont marquées d'une croix rouge. Une coche verte signifie que c'est la seule méthode possible. Par exemple, une adresse MAC ne peut pas être générée, ni être la même pour tous les postes client. Elle doit être saisie manuellement ou chargée depuis un fichier.
