Signification des couleurs des symboles

La couleur du symbole indique l'état de l'installation du poste client.
  • rouge: Le poste client a été ajouté et la détection du matériel automatique n'est pas encore terminée.
  • jaune: Vous pouvez formater et partitionner le poste client, le système de base et son interface graphique seront installés automatiquement.
  • vert: Le système de base du poste client est installé et vous pouvez y ajouter les logiciels supplémentaires.
  • bleu: Le poste client est en cours d'installation des logiciels supplémentaires.
  • orange: Le poste client est dans un état critique. Cela signifie qu'une erreur s'est produite pendant l'installation; elle doit être résolue par l'administrateur. Le centre de contrôle, vous propose plusieurs solutions afin de résoudre cet état critique.
  • blanc: Poste client modèle pour l'installation de masse à partir duquel les configurations seront transmises aux autres postes client.
  • Bogue: Indique que le poste client se trouve en mode de débogage.
  • gris: Ce client, purement virtuel, permet de sauvegarder les propriétés du client (IP, nom du client, MAC), dans le but qu'elles ne soient pas utilisées accidentellement.
