Postes client unitaires / paramètres du réseau

Ici, il vous est possible d'assigner des paramètres réseau spécifiques à des machines individuelles ou bien de réserver des valeurs spécifiques qui ne seront alors plus disponibles pour des postes client m23, ainsi il ne sera plus possible d'assigner à tort ces valeurs à un poste client m23.

  • Réserver le nom de poste client / l' IP / l'adresse MAC: En utilisant la fonction Réserver le nom de poste client / l' IP / l'adresse MAC, vous pouvez réserver un nom de poste client, une adresse IP ou une adresse MAC (par exemple l'adresse IP d'une imprimante réseau), afin que cette valeur ne puisse pas être utilisée à tort par des postes client m23. Entrez l'adresse IP, l'adresse MAC ou le nom que vous souhaitez réserver dans les champs ''Nom du poste client'', ''Adresse IP'' ou ''Adresse MAC'', puis cliquez sur ''Réserver le nom de poste client / l' IP / l'adresse MAC''. L'utilisation de la fonction ''Réserver le nom de poste client / l' IP / l'adresse MAC'' nécessite au moins une valeur dans un des champs marqués par 1.

  • Ajouter des paramètres au server DHCP: Si vous souhaitez assigner des paramètres réseau à une machine (externe) en utilisant DHCP, entrez les valeurs désirées dans les champs ''Nom du poste client'', ''Adresse IP'', ''Masque réseau'', ''Adresse MAC'' et ''Passerelle'', puis cliquez sur ''Ajouter des paramètres au server DHCP''. Tous les champs marqués par 2 sont nécessaires à cette fonction.
