Exigences pour m23

Pour pouvoir intégrer le système de distribution de logiciel de m23 dans votre réseau, il vous faut du suivant:
  • serveur m23
  • client(s) m23
  • accès à l'internet
  • ordinateur avec un navigateur web
  • CD d'installation du serveur m23
