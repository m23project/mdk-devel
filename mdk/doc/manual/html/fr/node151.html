Création d'un média d'amorçage

Ce dialogue vous permet de télécharger les fichiers ISO des médias d'amorçage pour les clients m23. Vous pouvez les graver sur un CD/DVD ou les utiliser directement depuis le média virtuel du logiciel de virtualisation. Si votre serveur m23 comporte un graveur CD/DVD vous pouvez graver directement un média d'amorçage avec celui-ci.
