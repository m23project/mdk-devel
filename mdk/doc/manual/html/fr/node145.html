Télécharger des outils complémentaires pour les postes client

Dans la liste ''Outils complémentaires pour VirtualBox téléchargeables'' s'affichent tous les outils complémentaires (depuis la version 2.0.0) actuellement téléchargeables sur la page web de VirtualBox. Les versions déjà téléchargées sur le serveur m23 sont marquées par ''Déjà téléchargé''. Sélectionnez les versions d'outils complémentaires pour le poste client VirtualBox correspondant à la version de l'hôte. Cliquez sur ''Télécharger'', après avoir choisi. Dans l'affichage sous ''État du téléchargement'', vous pouvez suivre le téléchargement de la version choisie. De temps en temps, cliquez sur ''Actualiser'' pour actualiser l'affichage.
