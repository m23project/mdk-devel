Note: UEFI

Si vous désirez démarrer votre client en utilisant l'UEFI, Vous devez sélectionner ''UEFI x86_64''. L'architecture sera automatiquement paramétrée pour un système 64 bits, puisque m23 ne supporte l'UEFI que pour les systèmes 64 bits. Si votre client ne supporte pas l'amorçage réseau en mode UEFI, Vous pouvez utiliser un DVD d'amorçage pour architecture 64 bits (Voir l'item ''Création d'un média d'amorçage'' du menu).
Chargeur d&apos;amorçage: Choisissez quel chargeur d'amorçage/boot loader vous souhaitez utiliser pour l'amorçage du noyau Linux et des autres systèmes d'exploitation eventuellement installés.
