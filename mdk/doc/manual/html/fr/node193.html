Changer le nombre de postes client

Vous pouvez changer le nombre de postes client en entrant le nouveau nombre dans Nombre de postes clients, puis cliquez sur Actualiser pour en diminuer le nombre ou sur Générer à nouveau les valeurs pour en augmenter le nombre.
Dans ce dernier cas, les programmes vont générer les valeurs nécessaires après que vous ayez cliqué sur Générer à nouveau les valeurs et tenter de lire les autres valeurs du fichier de données. Si le fichier de données ne contient pas assez de valeurs, vous devez les entrer manuellement dans la table. La génération d'un nombre important de valeurs peut sans doute durer un long moment, par exemple si les adresses IP sont en cours de génération et vous avez choisi de contacter les adresses IP avant leur utilisation.
Enfin, cliquez sur Enregistrer pour commencer l'installation des postes client.
