Configuration dynamique du serveur DHCP

Chaque fois l'état de démarrage par réseau d'un poste client est changé (activé / désactivé), le script ''/m23/bin/externalDHCPControl.sh'' sera exécuté. Vous pouvez adapter le script selon vos besoins afin de configurer dynamiquement un serveur DHCP existant. Trouvez plus d'information dans les commentaired dans le fichier script.
