share-online.biz

Si vous avez choisi ''share-online.biz'', vous devez entrer les valeurs suivantes:
  • Utilisateur: C'est le nom d'utilisateur, avec lequel vous êtes registrés chez share-online.biz.
  • Mot de passe: Entrez le mot de passe correspondant.
  • Veuillez répéter le mot de passe: Resaisissez-le pour confirmation.
  • Clé GPG: Sélectionnez la clé GPG, avec laquelle vous souhaitez chiffrer vos archives de sauvegardes avant le transfert.
