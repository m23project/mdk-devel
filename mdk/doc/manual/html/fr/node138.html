Créer une machine virtuelle

Ce dialogue vous permet de créer des postes client m23 virtuels (Machine Virtuelle). Les machines virtuelles se partagent les ressources du système hôte et se comportent comme des postes client m23 normaux sur le réseau. Afin de créer une nouvelle machine virtuelle sur l'hôte, le système hôte doit être démarrée et le paquet de virtualisation m23 doit y avoir été installé, (si la virtualisation intégrée VirtualBox est utilisée)(pour plus d'informations, voir ci-dessous).
