Remarque concernant les clés GPG

Si la clé GPG que vous souhaitez utiliser pour le chiffrement des données n'est pas dans la liste, cliquez sur Gérer les clés GPG.

  1. Sélectionnez la méthode de sauvegarde de données dans Service en ligne de sauvegarde de fichiers. vous avez le choix entre:
    • -: Cette option copie l'archive de sauvegarde uniquement sur le disque dur du serveur m23. Vous devriez utiliser cette option seulement si vous ne souhaitez absolument pas utiliser d'autre serveur pour la sauvegarde de vos archives. Dans le cas d'une perte de données, par exemple par défaillance du disque dur du serveur m23, le serveur m23 ne pourra pas être restauré. Vous pouvez copier les fichiers de sauvegarde, qui se trouvent dans le répertoire 
      /m23/root-only/serverBackups/
      , manuellement sur un média externe.
    • scp: Au moyen de Secure Copy vous pouvez sauvegarder les archives de sauvegarde sur un quelconque ordinateur accessible par son serveur SSH. Cette option s'impose si vous disposez d'un serveur SSH pouvant accueillir les sauvegardes. Si les données de ce serveur sont également sauvegardées par un cycle de sauvegarde, cette option est une possibilité très sûre pour la protection de votre serveur m23.
    • share-online.biz: Si vous sélectionnez cette option, le serveur m23 va transférer vos archives de sauvegarde chiffrées sur un One-Click-Hoster. Si vous n'avez pas de serveur pour copier l'archive de sauvegarde de votre serveur m23, cette option pourrait vous intéresser. Vos données de sauvegarde seront protégées par le chiffrement contre l'accès non autorisé. Le téléchargement peut durer quelque temps.
  2. Cliquez sur Changer après avoir choisi.
  3. Dépendant du service en ligne de sauvegarde de fichiers vous devrez fournir différentes indications afin de pouvoir utiliser ce service.
