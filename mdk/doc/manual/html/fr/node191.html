Architecte de sources de paquets (Sélectionner les sources de paquets et les paquets)


Choisissez l'origine des sources de paquets logiciels pour l'installation. Le dialogue se décompose en deux parties:
  1. D'abord, sélectionnez les sources de paquets.
    • Charger une source de paquets: Pour le chargement, choisissez la source de paquets dans la liste en haut à droite, puis cliquez sur Charger, elle sera chargé dans l'éditeur.
    • Effacer une source de paquets: Pour effacer la source de paquets chargé dans l'éditeur, cliquez sur le bouton Effacer et confirmez l'effacement au dialogue suivant.
    • Enregistrer une source de paquets: La source de paquets chargée dans l'éditeur sera toujours celle qui sera enregistrée, indépendant de celle sélectionnée dans la liste. Choisissez d'abord la distribution à laquelle la liste des paquets doit s'appliquer. Après avoir choisi la distribution, il est possible que vous deviez entrer des valeurs supplémentaires. De plus, sélectionnez les interfaces graphiques supportés dans la liste des sources de paquets sous le champ de l'éditeur et la/les architecture(s) processeur(s) supportée(s) sous ''Architecture du processeur''. Trouvez un nom pour la liste de paquets et entrez-le dans le champ de saisie sous Nom de source de paquets. Si vous avez chargé une source de paquets, le nom sera entrée automatiquement. Si vous voulez, vous pouvez entrer un commentaire dans le champ Description; puis, cliquez sur Enregistrer.
    • Tester une source de paquets: Après avoir entré une source de paquets, vous devriez tester si toutes les sources fonctionnent correctement. Après un clic sur le bouton Tester, le programme tente de télécharger les descriptions des paquets puis lance un test. Après l'achèvement de ce test un rapport apparaîtra; cela peut prendre quelques minutes jusqu'à son apparission. La vitesse dépend de votre connection internet et de la vitesse des serveurs de sources de paquets utilisés.
    • Entrer le site miroir: Si vous souhaitez utiliser un site miroir différent du site miroir standard est ftp.debian.org pour l'installation du système de base, entrez-le sur une nouvelle ligne dans la source de paquets comme décrit ci-après: 
      #mirror: [URL des paquets]
      . URL des paquets signifie le protocole (http, ftp, ...), le nom d'hôte ou l'adresse IP et le répertoire, où les paquets peuvent être trouvés. Une ligne valide pourrait se présenter comme cela, par exemple: 
      #mirror: http://192.168.7.14/debianCDs
    Ensuite,la liste affichée dans la fenêtre Sources de paquets sera utilisée.
  2. Dans la partie inférieure du dialogue, entrez les noms des paquets que vous souhaitez enregistrer dans votre source de paquets. Les noms des paquets doivent être séparés par des espaces.
    • Charger / Effacer: Si vous avez déjà enregistré une liste de paquets, vous pouvez la charger dans la fenêtre de l'éditeur ou l'effacer en cliquant sur Charger ou Effacer après l'avoir sélectionnée dans le champ Liste de paquets.
    • Ajouter: Vous pouvez ajouter les paquets dans la liste de paquets à ceux dans l'éditeur en cliquant sur Ajouter.
    • Enregistrer: Si vous souhaitez enregistrer la liste affichée dans l'éditeur, entrez un nom dans le champ de saisie Liste de paquets, puis, cliquez sur Enregistrer
    • Ajouter la liste de paquets d&apos;un poste client spécifique: Cette fonction ajoute les noms des paquets installés sur le poste client sélectionné à la liste de paquets affichée dans l'éditeur. Pour pouvoir ajouter tous les paquets d'un poste client, sélectionnez le poste client dans la liste (qui contient seulement les postes client sur lesquels il y a des paquets installés) et ensuite, cliquez sur Ajouter. Il est également possible d'ajouter des paquets provenant de plusieurs postes clients - en ce cas, les paquets en double ne seront affichés qu'une seule fois.
  3. Si vous activez la fonction Télécharger les paquets de base tous les fichiers nécessaires à l'installation du système de base (sauf le noyau du poste client) seront téléchargés.
  4. Enfin, cliquez sur Étape suivante (Lancer le téléchargement).
