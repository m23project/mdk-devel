Concernant les fichiers image

Dans la liste des sources de paquets, sélectionnez l'image pour l'installation du fichier image désiré.
Si la liste vous n'offre pas d'interfaces graphiques, néanmoins confirmez par un clic sur "Choisir" pour pouvoir procéder à la prochaine partie du dialogue.
Maintenant, selectionnez la partition où vous voudriez que le MBR (Master Boot Record) soit installé et assignez les fichiers image (ou rien) aux partitions de la machine destinataire.
Vous pouvez choisir d'installer sur le poste client le MBR (Master Boot Record) à partir d'un fichier créé précédemment ou bien le MBR générique de m23. Pour ce faire, choisissez sous Installer un MBR à partir d&apos;une image ou installer un MBR générique. entre le nom du fichier ou MBR générique.
Si vous choisissez une distribution correspondante à celle du fichier image sous "Configurer l&apos;image comme", m23 essaiera de configurer la machine tant qu'elle peut être administrée par m23.
