Désinstaller des paquets:

  1. Choisissez Combinaisons de paquets ou Paquets normaux.
  2. Entrez le mot recherché pour votre paquet dans le champ de saisie Recherche des paquets ou laissez le champ vide si vous souhaitez afficher tous les paquets d'un type de paquets installés sur le poste client.
  3. Sous Paquets trouvés, vous apparaissent tous les paquets correspondants à vos critères de recherche.
  4. Cochez la case pour les paquets dont vous souhaitez la désinstallation.
  5. Présélectionnez les paquets à désinstaller en cliquant sur Noter.
  6. Répétez les étapes 1 à 5 selon vos besoins.
Après que vous ayez présélectionné les paquets pour la désinstallation, une coche verte indique les paquets à installer, une croix rouge indique ceux à effacer du client. Vous pouvez effacer la liste de ces paquets en cliquant sur Tout Annuler ou sélectionner des paquets et les effacer après avoir cliqué sur Annuler la sélection.
Confirmez votre choix en cliquant sur Désinstaller les paquets sélectionnés.
