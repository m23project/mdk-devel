BitTorrent

Lorsque vous souhaitez distribuer des fichiers volumineux à plusieurs ordinateurs clients tout en réduisant la charge induite sur le serveur, vous pouvez utiliser la fonctionnalité BitTorrent. BitTorrent rend possible le partage des fichiers déjà téléchargés sur des clients m23 pour les autres clients, (dans le cas idéal) le serveur m23 ne transférera le fichier qu'une seule fois.
Pour utiliser Bittorrent, suivez ces étapes:
  1. Copiez le(s) fichier(s) à transférer sur les clients dans le répertoire ''/m23/data+scripts/extraDebs/'' de votre serveur m23.
  2. Cliquez sur le bouton ''Création du .torrent'' sous le fichier correspondant.
  3. Dans vos propres scripts, utilisez la fonction m23 
    BT\_dlFile($torrentFileName, $DestinationDirectoryOnClient)
    .
