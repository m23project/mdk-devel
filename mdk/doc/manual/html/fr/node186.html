Notez: Combinaisons de paquets

Si vous cliquez sur Combinaisons de paquets, vous pouvez décider l'installation ou la désinstallation des paquets sur le poste client. De plus, vous pouvez conserver les actions enregistrées dans la combinaison de paquets. Ceci vous permet d'utiliser des travaux d'installation et de désinstallation dans une combinaison particulière. Pour cela, choisissez l'action souhaitée dans la liste.
