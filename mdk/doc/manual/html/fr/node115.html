Autre serveur DHCP

  1. Choisissez ''PXE'' dans ''Type de boot'' lorsque vous ajoutez un nouveau client m23 par l'interface web. Saisissez le nom de l'ordinateur à assigner au client m23 par le serveur DHCP dans ''Nom du poste client''.
  2. Ajustez votre serveur DHCP de sorte qu'il assigne le nom du poste client indiqué dans l'interface web m23 au poste client m23.
  3. Le nom du noyau PXE, transmis par votre serveur DHCP au poste client, doit être pxelinux.0. Le serveur TFTP est le serveur m23.
  4. Configurez le serveur DHCP de manière à ce qu'il fournisse l'adresse IP du serveur m23 comme adresse d'origine pour les clients m23.
  5. Si malgré cela, votre serveur DHCP ne fonctionne toujours pas après l'ajustement des paramètres, vous devez installer un serveur TFTP sur l'ordinateur hébergeant votre serveur DHCP. La configuration de base du serveur TFTP peut s'inspirer des fichiers de configuration du répertoire /m23/tftp/.
