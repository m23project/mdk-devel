Suchen und ersetzen

  • Suchen: Strg + F (PC), Command + F (Mac)
  • Nächstes Vorkommen suchen: Strg + K (PC), Command + G (Mac)
  • Voriges Vorkommen suchen: Strg + + K (PC), Command + + G (Mac)
  • Suchen und ersetzen: Strg + H (PC), Command + Option + F (Mac)
  • Zu einer bestimmten Zeilennummer springen: Strg + L (PC), Command + L (Mac)
