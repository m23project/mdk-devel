Xhprof-Einstellungen

Auf dieser Seite können Sie Ihren m23-Server mittels Xhprof profilieren, um das Laufzeitverhalten (CPU- und Speichernutzung) zu analysieren.

Unter ''Xhprof (de)aktivieren'' legen Sie durch Auswahl von ''Aktivieren'' fest, daß Xhprof verwendet werden soll. Über ''Deaktivieren'' können Sie das Profiling wieder deaktivieren. Bei aktivem Profiling wird eingeblendet, wie lange die Erstellung der gerade angezeigten Seite benötigte.
Möchten zusätzlich die Ergebnisse speichern, um (nachträglich) das Laufzeitverhalten der einzelnen Funktionen zu analysieren, so wählen Sie ''JA'' unter ''Resultate speichern''. Nach der Aktivierung wird für jede generierte Seite eine eigenen Profiling-Datei angelegt. Die Profiling-Dateien finden Sie unter '' 
/var/cache/xhprof/runs
''. Zum Deaktivieren wählen Sie entsprechend ''NEIN''.
Die aktuelle Auswahl speichern Sie anschließend mit einem Klick auf ''Ändern''.
Bereits gespeicherte Profilings können Sie mit einem Klick auf ''Resultate anzeigen'' abrufen und mit einem Klick auf das rote X vor dem jeweiligen Profiling auch wieder löschen.
