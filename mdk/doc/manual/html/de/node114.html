Hinweis zur Verwendung von JavaScript

Die Aktivierung von JavaScript ist optional. Bei aktiviertem JavaScript stehen einige zusätzliche Komfortfunktionen (z.B. das automatische Aufklappen der Menüpunkte oder die Kennzeichnung der Partition unter dem Mauszeiger) zur Verfügung. Aber auch ohne JavaScript können alle nötigen Funktionen genutzt werden.
