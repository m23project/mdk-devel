Hinweis zu UEFI

Verwendet Ihr m23-Client UEFI, so muß zuerst eine EFI-Boot-Partition angelegt werden. Die Position und Größe sind bei dieser vorgegeben und kann nicht geändert werden.
Partition formatieren Hiermit formatieren Sie die aktuell ausgewählte Partition. Wählen Sie hierzu unter ''Typ'' das gewünschte Dateisystem und klicken anschließend auf ''Formatieren''.
