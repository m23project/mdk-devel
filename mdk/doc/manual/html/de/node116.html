Dynamische Konfiguration von DHCP-Servern

Bei jeder Netzwerkboot-Status-Änderung (Aktivierung bzw. Deaktivierung) eines Clients wird das Skript ''/m23/bin/externalDHCPControl.sh'' aufgerufen. Sie können dieses nach Ihren Wünschen anpassen, um damit einen vorhandenen DHCP-Server dynamisch zu konfigurieren. Weitere Informationen entnehmen Sie bitte den Kommentaren innerhalb des Skriptes.
