Hinweis: UEFI

Falls Sie Ihren Client über UEFI starten möchten, wählen Sie ''UEFI x86_64'' aus. Hierbei wird die Rechnerarchitektur automatisch auf 64 Bit gesetzt, da m23 UEFI nur auf 64-Bit-Systemen unterstützt. Sollte Ihr Rechner kein Netzwerkbooten im UEFI-Modus unterstützen, so können Sie das 64-Bit-Boot-ISO verwenden (zu finden z.B. im Menü unter ''Boot-CD brennen'').
Bootloader: Wählen Sie hier, welchen Bootloader/Bootmanager/Urlader Sie zum Starten des Linux-Kernels und evtl. anderen, bereits installierten Betriebssystemen verwenden möchten. Sie haben die Wahl zwischen LILO (LInux LOader) und GRUB (GRand Unified Bootloader).
