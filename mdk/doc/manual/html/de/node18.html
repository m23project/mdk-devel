Die m23-Server-Installations-CD

Um einen m23-Server aufzusetzen, benötigen Sie die m23-Server-Installations-CD die es zum kostenlosen Download auf http://m23.sf.net gibt. Brennen Sie die ISO-Datei einfach mit einem Brennprogramm und booten Sie danach den Server davon.
