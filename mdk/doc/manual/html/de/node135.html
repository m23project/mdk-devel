Mehrere Clients bearbeiten

Wählen Sie mehrere Clients dadurch aus, daß Sie den Haken in den Spalte mit den gewünschten Clients setzen.
Danach können Sie verschiedene Aktionen mit allen gewählten Clients durchführen:
  • Aus Gruppe löschen: Wählen Sie hierzu die Gruppe aus, aus der die Clients gelöscht werden sollen. Sollte ein Client nicht in der angegebenen Gruppe sein, so wird mit diesem keine Aktion durchgeführt.
  • Zur Gruppe hinzufügen: Wählen Sie hierzu die Gruppe aus, zu der die Clients hinzugefügt werden sollen. Ist ein Client bereits in dieser Gruppe vorhanden, so wird an seiner Gruppenzugehörigkeit nichts geändert.
  • Löschen: Löscht die gewählten Clients
  • In Gruppe verschieben: Verschiebt einen Client in eine Gruppe.
  • Wiederherstellen: Installiert den Client komplett neu.
