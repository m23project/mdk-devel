Hinweis

Die m23-Server-Installations-CD enthält ein Skript, um Ihre vorige m23 Installation auf die Version zu updaten, die auf der CD enthalten ist. Gehen Sie zum Updaten Ihres m23 Servers wie folgt vor:
  1. Öffnen Sie eine Konsole
  2. Legen Sie die m23-Server-Installations-CD ein
  3. Mounten Sie die CD (z.B.: 
    mount /mnt/cdrom1
    )
  4. Wechseln Sie in das Mountverzeichnis der CD (z.B.: 
    cd /mnt/cdrom1
    )
  5. Führen Sie das Update aus: 
    bash updatem23.sh
