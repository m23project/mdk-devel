Tricks

  • Durch Klicken auf das Statussysmbol kann der aktuelle Status des Clients geändert werden.
  • Allerdings sollten Sie den Client-Status nur ändern, wenn Sie wissen, was Sie tun. Der Debug-Modus kann durch einen Klick auf das Laus-Icon (de)aktiviert werden.
