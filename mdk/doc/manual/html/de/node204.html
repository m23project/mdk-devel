Textauswahl

  • Aktuelle Zeile löschen: Strg + D (PC), Command + D (Mac)
  • Kompletten Quelltext auswählen: Strg + A (PC), Command + A (Mac)
  • Worte im Text ausgehend vom Textcursor auswählen: + &larr; bzw. (PC), + &larr; bzw. (Mac)
  • Zeile über bzw. unter aktueller wählen: + bzw. (PC), + bzw.
  • Text zum Zeilenanfang bzw. Zeilenende ausgehend vom Textcursor auswählen: + Pos1 bzw. Ende (PC), + Home bzw. Ende
