Paßwort

Ist der Client noch in einem frühen Installationsstadium, so lautet das Root-Paßwort: ''NETROOTPWD''
Später ist es das Paßwort, das Sie beim Anlegen des Clients vergeben haben.
Sollte Ihr Browser das SSH-Protokoll direkt unterstützen, so können Sie die Verbindung durch Anklicken dieses Links herstellen: ssh://root@CLIENTIP
Sonst öffnen Sie bitte eine Konsole und geben den Befehl: ssh -o UserKnownHostsFile=/dev/null -l root CLIENTIP Um in die aktuelle Client-Bildschirmausgabe zu lesen oder in die laufende Installation einzugreifen benutzen Sie den Befehl: screen -x m23install
