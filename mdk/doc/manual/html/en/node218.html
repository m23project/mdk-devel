Hint

  • Pool type: The type selects the media the packages should be fetched from. ''Download pool'': This variant downloads all packages from HTTP or FTP servers that can be used for package installation.
  • Architecture: Choose here the architecture for the packages to download. Here the architecture of the clients is meant and not the architecture of the m23 server.
  • Package source: A package source is a directory (with subdirectories) that contains software packages and index files. The index files are storing information about the software packages and are needed to make the packages available for installation.
