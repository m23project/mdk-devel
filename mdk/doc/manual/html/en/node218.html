Change the number of clients

You can change the number of clients by entering a new value in the field ''Number of clients''. Click on ''Refresh'' if you want to lower the amount or on ''Regenerate values'' to increase.
''Regenerate values'' can generate new values or read additional lines from the database file, whereas ''Refresh'' can only delete clients. If there are not enough lines in the database file, these values have to be entered by hand. The generation of a huge amount of values can take a long time. E.g. ''pinging'' of clients.
To start the installation click on ''Save''.
