Add update group

To be able to use the automatical update functionality, you will need at least one update group. Create the group at ''Add update group''. At ''Simultaneous updates'', enter the maximum number of clients that should be updated simultaneously. At ''Type of update'', you can select to use either Normal update (to update installed packages, and only install new packages when these are required) or Full update (to update installed packages, install new packages, and uninstall packages). Then click on ''Add update group''.
