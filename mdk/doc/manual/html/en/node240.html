There are 3 methods

  • Enter: The values are entered by hand or read from a file.
  • Generate: The values are generated automatically. E.g. it is possible to generate client names with incrementing numbers or IP adressess with unused IP numbers.
  • Keep: The value from the ''master client'' is kept on all clients.
You don't have the possiblillity to select all 3 methods on every client option. Not selec options are marked with a red cross. A green check shows that this method is the only one that works for this client option. E.g. it is not possible to generate MAC addresses nor to keep a MAC for all clients. MAC adresses always have to be entered by hand or read from a file.
