Completion of the installation

Hint: You don't need a monitor for normal operation except if you want to use the server as a workstation, too. The server is a Debian GNU/Linux system, this makes it possible to log in as ''root'' and install additional software in the Debian-way. For information on how to install software under Debian, have a look at www.debian.org.
You can start setting up your m23 clients now!
