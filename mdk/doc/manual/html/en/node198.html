Text selection

  • Delete current line: Ctrl + D (PC), Command + D (Mac)
  • Select all: Ctrl + A (PC), Command + A (Mac)
  • Select text beginning at current cursor position: + &larr; or (PC), + &larr; or (Mac)
  • Select line above or below current line: + or (PC), + or
  • Select text until start or end of current line beginning at current cursor position: + Pos1 or End (PC), + Home or End
