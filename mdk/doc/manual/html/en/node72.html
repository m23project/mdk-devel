Setting up a VM host

If you would like to use this m23 client as a host for virtual m23 clients, click on "Install virtualisation host". The virtualisation software (if available for the installed distribution) will then be installed on the m23 client and the client will be added to the list of VM hosts. After the installation is completed, you can create and start virtual m23 clients on your new host by clicking on the menu point "Create VM".
