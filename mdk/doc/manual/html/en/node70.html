Work on multiple clients

Select the clients by checking the boxes in the client lines.
Now you can work on all selected clients::
  • remove from group: Select the group, all clients should be deleted from. Nothing is done, if a client is not in the selected group.
  • add to group: Select the group, all clients should be added to. Nothing is done, if a client already is in the selected group.
  • move to group: Moves the clients to another group.
  • Recover: Completely reinstalls the clients.
  • Delete: Deletes the selected clients.
