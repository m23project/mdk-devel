Alternatively: Create boot media

When it is not possible to change the configuration of your DHCP server, you can still start your m23 client using a USB / optical boot medium, and then on the client's command line, run the script '' 
fetchJobManual
'' to start the client's install process. Further information about boot medium creation is available from ''Create boot media'' in the menu.
