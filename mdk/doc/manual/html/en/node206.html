Step by step

  1. Enter the separator character in the separator field and adjust the column amount if needed.
  2. Click on ''Refresh''. The first line of the database file is now separated and shown in the client option fields.
  3. Assign the separated parts from the database file to the client options by selecting the client option names from the selection list.
  4. End the dialog with a click on ''Save''.
