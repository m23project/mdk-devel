Hint for imaging

Select "imaging" as package sources to install images.
You can choose if you want to install a previously created MBR file or the generic bootloader of m23 to the the client's MBR (Master Boot Record). Select the file name or "Generic MBR" from the "Install the generic MBR or the MBR from an image" list.
