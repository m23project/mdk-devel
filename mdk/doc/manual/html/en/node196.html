Find and replace

  • Search: Ctrl + F (PC), Command + F (Mac)
  • Find next occurance: Ctrl + K (PC), Command + G (Mac)
  • Find previous occurance: Ctrl + + K (PC), Command + + G (Mac)
  • Find and replace: Ctrl + H (PC), Command + Option + F (Mac)
  • Jump to line number: Strg + L (PC), Command + L (Mac)
