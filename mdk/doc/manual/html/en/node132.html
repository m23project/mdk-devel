Hint for imaging

Select ''imaging'' as package sources to install images.
If the drop down list does not offer any user interfaces, nevertheless confirm this by clicking on ''Select'' to get to the next part of the dialogue.
Now choose the partition where you would like the MBR (Master Boot Record) to be installed and assign the desired image file (or none) to the partitions of the targeted machine.
You can choose if you want to install a previously created MBR file or the generic bootloader of m23 to the the client's MBR (Master Boot Record). Select the file name or ''Generic MBR'' from the ''Install the generic MBR or the MBR from an image'' list.
If you choose a distribution consistent with the image file's distribution at ''Configure image like'', m23 will try to configure the machine so that it can be administered using m23.
