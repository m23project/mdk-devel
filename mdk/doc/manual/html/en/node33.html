Manage administrators

You can add and delete administrators in this dialog. Administrators are people with a certain login name and password for the m23 administration interface on the current m23 server.

  • Add administrators by entering the login name and the desired password.
  • Delete an administrator by clicking the ''Delete'' link after the name. The login name can't be deleted, if there is only one administrator
