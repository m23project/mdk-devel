Hint

The m23 server installation CD contains a script that lets you update the currently installed m23 version to the version on the CD. To update your system, proceed as follows:
  1. Open a shell
  2. Insert the m23 server installation CD
  3. Mount the CD (e.g.: 
    mount /mnt/cdrom1
    )
  4. Change to the mount directory of the CD (e.g.: 
    cd /mnt/cdrom1
    )
  5. Update your system and type: 
    bash updatem23.sh
