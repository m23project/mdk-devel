Make boot CD

Hier können Sie eine Client-Boot-CD mit einem in dem Server eingebauten Brenner erstellen.
  1. Select the desired burner (if there should be installed more than one burner in your server) and click on ''Select''.
  2. If you want to use a CD-RW, you have to blank it before usage. This can be done in the ''Blank CD-RW'' box. You can choose if you want to blank a little part of the RW only (this should be ok for most purposes) or the whole RW. Select ''fast'' for blanking the little part or ''all'' for the whole RW.
  3. The last box ''Make boot CD'' is used to burn the client boot CD. Simply click (after choosing the burn speed)on the ''Burn'' button.
The box ''Burner status'' shows information about the last blanking or burning action and the current state of the burner.
