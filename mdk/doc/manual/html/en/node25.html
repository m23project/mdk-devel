Adding a client

This section shows the abstract procedure of adding a client. For the exact procedure you should have a look at the specific chapter.

  • Connect the client to your network to make the client able to connect to the server.
  • If your client doesn't have a network card with PXE or Etherboot you need to create a boot disk or CD.
  • Start the client and take down the MAC address (e.g. 00:45:23:3A:96:F3).
  • Enter the needed values and the MAC address to add the client (as described in the chapter ''Manage your clients'').
  • Reset the client if it doesn't boot automatically. Remember to insert the boot disk or boot CD (if needed).
  • The client should boot now and send hardware and partition information to the m23 server.
  • In the web interface you can examine these values. Continue with the chapter ''Partition and format the clients'' and ''Installation of the operating system'' afterwards.
  • To install additional software have a look at the ''Install/deinstall packages'' chapter.
