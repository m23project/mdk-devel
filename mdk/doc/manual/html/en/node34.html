Step by step

  • Enter the ''URL with update information:'' or leave it as it is, if you don't have another update URL.
  • Click on ''Fetch update information''. Now you will see the version number of the new release (if there is one) and information about the release in the ''Update information'' box.
  • To install the update, click on ''Update server''. The update will be processed in the background and you will be able to work with the m23 server as usual.
