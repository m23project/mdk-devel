share-online.biz

You need to enter the following values if you prefer the ''share-online.biz'' variant:
  • User: This is the user name you used during the registration process on share-online.biz.
  • Password: Enter the according password here.
  • Password (repeated): Repeat it here, to avoid mistyping.
  • GPG key: Choose the GPG key for encrypting the backups before copying.
