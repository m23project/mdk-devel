Hint

All in the file stored values have to be valid and there are an additional requirements for special options:
  • Store login information local on the client.: Set it to ''yes'' if you want to activate this option. All other values disable it.
  • LDAP: Here three different values are possible:
    • ''none'': Don't use LDAP
    • ''read'': Read login information from selected LDAP server.
    • ''write'': Store login information on the selected LDAP server.
